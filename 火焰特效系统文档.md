# 火焰特效系统文档

## 一、概述

火焰特效系统由两个核心类组成：
- `FireEffect`：负责3D粒子系统的物理模拟和颜色生命周期管理
- `Flame2DEffect`：负责将3D粒子投影到2D屏幕，并绘制火焰轮廓

两者通过单例模式协同工作，实现"上M下U"形状的火焰2D轮廓。

## 二、FireEffect（3D粒子系统）

### 2.1 类结构

```typescript
export class FireEffect extends BaseEffect
```

继承自 `BaseEffect`，管理火焰粒子的完整生命周期。

### 2.2 粒子数据结构

```typescript
class FireParticle {
  position: THREE.Vector3      // 世界坐标位置
  velocity: THREE.Vector3      // 速度向量
  color: THREE.Color           // 当前颜色
  size: number                // 粒子大小
  lifetime: number           // 剩余寿命
  maxLifetime: number        // 最大寿命（用于计算生命因子）
  seed: number               // 随机种子（用于湍流相位）
}
```

### 2.3 粒子发射

```typescript
private emitParticle()
```

**发射位置**：
- X/Z范围：`(Math.random() - 0.5) * 0.8`，即 [-0.4, 0.4]
- Y = 0（地面）

**初始速度**：
- 半球发射模式，phi（方位角）0-360°，theta（极角）0-90°
- 速度大小：1.2 + Math.random() * 0.4，即 [1.2, 1.6]

**初始颜色**：白热色 `(1.0, 0.9, 0.6)`

**粒子大小**：0.08（初始值，会随生命周期变化）

**寿命**：1.0 ~ 1.8 秒

### 2.4 物理模拟（onUpdate）

每帧更新包含以下物理计算：

**1. 上升浮力**
```typescript
let upwardForce = 1.2 * (1 - height / 3.0);
if (upwardForce < 0.2) upwardForce = 0.2;
```
随高度递减，最低保持0.2，模拟热空气上升后冷却。

**2. 水平扩散**
```typescript
let spreadStrength = 1.2 * Math.max(0, 1 - height / 2.0);
spreadStrength += 0.3 * (1 - lifeFactor);
```
底部强，随高度衰减。生命后期略微增加扩散，模拟火焰消散。

**3. 湍流（让火焰摇曳）**
```typescript
const time = performance.now() * 0.001;
const phase = particle.seed * 100;
const turbX = Math.sin(time * 3.0 + phase) * 1.2;
const turbZ = Math.cos(time * 2.3 + phase) * 1.2;
```
使用正弦波 + 随机相位产生自然的摇曳效果。

**4. 顶部收缩力**
```typescript
if (height > 1.0 && radius > 0.1) {
  const inwardStrength = 0.5 * (height - 1.0);
  particle.velocity.x -= (particle.position.x / radius) * inwardStrength;
  particle.velocity.z -= (particle.position.z / radius) * inwardStrength;
}
```
让火焰顶部收窄，模拟火焰尖部变细的形态。

**5. 阻力**
```typescript
particle.velocity.multiplyScalar(0.99);
```
增大阻力，使火焰更高更飘。

### 2.5 颜色与大小生命周期

```typescript
if (lifeFactor > 0.7) {
  // 初始：红白色
  particle.color.setRGB(1.0, 0.8, 0.8);
  particle.size = 0.2; // 增大3D粒子大小
} else if (lifeFactor > 0.3) {
  // 中期：橙色
  particle.color.setRGB(1.0, 0.6, 0.0);
  particle.size = 0.25; // 增大3D粒子大小
} else {
  // 后期：红色
  particle.color.setRGB(0.8, 0.1, 0.0);
  particle.size = 0.15; // 增大3D粒子大小
}
```

生命因子从1→0，颜色从红白色→橙色→红色变化，大小也相应调整。

### 2.6 3D粒子渲染

使用自定义着色器实现圆形粒子：

```typescript
this.material = new THREE.ShaderMaterial({
  uniforms: {
    pointSize: { value: 9.0 } // 基本粒子大小
  },
  vertexShader: `
    uniform float pointSize;
    attribute float size;
    attribute vec3 color;
    varying vec3 vColor;
    
    void main() {
      vColor = color;
      vec4 mvPosition = modelViewMatrix * vec4(position, 1.0);
      gl_PointSize = pointSize * size * (100.0 / -mvPosition.z);
      gl_Position = projectionMatrix * mvPosition;
    }
  `,
  fragmentShader: `
    varying vec3 vColor;
    
    void main() {
      // 绘制圆形粒子
      float r = 0.5;
      vec2 cxy = 2.0 * gl_PointCoord - 1.0;
      float ll = dot(cxy, cxy);
      if (ll > 1.0) discard;
      
      // 径向渐变
      float alpha = 1.0 - ll;
      gl_FragColor = vec4(vColor, alpha);
    }
  `,
  transparent: true,
  blending: THREE.AdditiveBlending,
  depthWrite: false
});
```

**特点**：
- 圆形粒子，边缘平滑
- 径向渐变效果（中心亮，边缘暗）
- 透视校正的大小
-  additive 混合，营造火焰的亮度效果

### 2.7 传递给2D效果

```typescript
const particleData = this.particles.map(p => {
  const worldPos = p.position.clone();
  this.group.localToWorld(worldPos);
```

将粒子位置转换到世界坐标后，传递给 `Flame2DEffect` 进行2D渲染。

## 三、Flame2DEffect（2D轮廓渲染）

### 3.1 单例模式

```typescript
private static instance: Flame2DEffect | null = null;

public static getInstance(): Flame2DEffect {
  if (!Flame2DEffect.instance) {
    Flame2DEffect.instance = new Flame2DEffect();
  }
  return Flame2DEffect.instance;
```

使用单例确保只有一个canvas覆盖层，避免多个火焰实例产生重叠轮廓。

### 3.3 模板定义

**侧面模板（M形顶 + U形底）**：

```typescript
private sideTemplateNorm: { x: number; y: number }[] = [
  // 顶部 M 形山峰（y=0 顶部）
  { x: -0.25, y: 0.00 },   // 左峰
  { x: -0.10, y: 0.08 },   // 左谷
  { x: 0.00, y: 0.10 },    // 中峰最高
  { x: 0.10, y: 0.08 },    // 右谷
  { x: 0.25, y: 0.00 },    // 右峰
  // 中部过渡
  { x: 0.30, y: 0.30 },
  { x: 0.35, y: 0.50 },
  // 底部 U 形（y=1 底部）
  { x: 0.30, y: 0.80 },
  { x: 0.20, y: 0.90 },
  { x: 0.00, y: 0.95 },    // 底部中央略高（U 形）
  { x: -0.20, y: 0.90 },
  { x: -0.30, y: 0.80 },
  // 返回顶部左峰（闭合）
  { x: -0.35, y: 0.50 },
  { x: -0.30, y: 0.30 },
  { x: -0.25, y: 0.00 },
];
```

特点：
- y=0 是顶部（M形山峰）
- y=1 是底部（U形弧）
- x范围 [-0.5, 0.5]

**俯视模板（多瓣星形）**：

```typescript
const radialSegments = 60;
for (let i = 0; i <= radialSegments; i++) {
  const angle = (i / radialSegments) * Math.PI * 2;
  const r = 0.8 + 0.3 * Math.cos(3 * angle);  // 三瓣
  const x = Math.cos(angle) * r;
  const y = Math.sin(angle) * r;
  this.topTemplateNorm.push({ x, y });
}
```

### 3.4 视角选择逻辑

```typescript
isTopView = cameraDirection.y < -0.7;
```

只有当相机明显向下看（y方向分量 < -0.7）时，才使用俯视模板。其他情况使用侧面模板。

### 3.5 轮廓计算流程（computeTemplateContour）

**步骤1：计算粒子包围盒**
```typescript
let minX = Infinity, maxX = -Infinity, minY = Infinity, maxY = -Infinity;
for (const p of screenPoints) {
  if (p.x < minX) minX = p.x;
  if (p.x > maxX) maxX = p.x;
  if (p.y < minY) minY = p.y;
  if (p.y > maxY) maxY = p.y;
}
const width = maxX - minX;
const height = maxY - minY;
```

**步骤2：根据位置过滤颜色**
```typescript
const yThreshold = (minY + maxY) / 2;
const filtered = screenPoints.filter(p => {
  // 峰形区域（顶部，y值小 < yThreshold）：找红色
  // 弧形区域（底部，y值大 >= yThreshold）：找红白色
  if (p.y < yThreshold) {
    return isRed && !isOrange;
  } else {
    return isRedWhite && !isOrange;
  }
});
```

颜色判定规则：
- **红白色**（焰底）：r>0.9, g>0.7, b>0.7
- **红色**（外缘）：r>0.7, g<0.3, b<0.2
- **橙色**（中部）：r>0.8, g>0.4, g<0.8, b<0.3 → 排除

**步骤3：模板缩放**
```typescript
const contour = template.map(p => {
  let x = minX + (p.x + 0.5) * width;
  let y = minY + p.y * height;
  if (isTopView) {
    const centerX = (minX + maxX) / 2;
    const centerY = (minY + maxY) / 2;
    x = centerX + p.x * width / 2;
    y = centerY + p.y * height / 2;
  }
  return { x, y };
});
```

**步骤4：平滑曲线**
```typescript
const points3D = contour.map(p => new THREE.Vector3(p.x, p.y, 0));
const curve = new THREE.CatmullRomCurve3(points3D);
curve.curveType = 'centripetal';
curve.closed = true;
const smoothPoints3D = curve.getPoints(100);
```

使用Catmull-Rom样条曲线生成平滑轮廓，100个采样点。

### 3.6 渲染流程（render）

```typescript
private render()
```

1. **清空画布**：`this.ctx.clearRect(0, 0, this.width, this.height)`

2. **投影粒子到屏幕**：
```typescript
const ndc = p.position.clone().project(this.camera);
const screenX = (ndc.x + 1) / 2 * this.width;
const screenY = (1 - ndc.y) / 2 * this.height;
```

3. **绘制粒子**（默认关闭）：
```typescript
if (this.showParticles) {
  const size = Math.max(1, 4 / (pt.depth * 0.5 + 0.5));
  // 绘制带径向渐变的圆形粒子
}
```

4. **定时更新轮廓**（0.01秒间隔）：
```typescript
if (now - this.lastContourUpdate > this.contourUpdateDelay) {
  this.lastContourUpdate = now;
  this.cachedContour = this.computeTemplateContour(screenPoints);
}
```

5. **绘制轮廓**：
```typescript
this.ctx.beginPath();
this.ctx.moveTo(this.cachedContour[0].x, this.cachedContour[0].y);
for (let i = 1; i < this.cachedContour.length; i++) {
  this.ctx.lineTo(this.cachedContour[i].x, this.cachedContour[i].y);
}
this.ctx.closePath();
this.ctx.shadowBlur = 6;
this.ctx.shadowColor = this.contourColor;
this.ctx.strokeStyle = this.contourColor;
this.ctx.lineWidth = this.contourWidth;
this.ctx.stroke();
```

## 四、EffectManager集成

### 4.1 EffectManager结构

```typescript
export class EffectManager {
  private static instance: EffectManager;
  private activeEffects: BaseEffect[] = [];

  private constructor() {}

  public static getInstance(): EffectManager {
    if (!EffectManager.instance) EffectManager.instance = new EffectManager();
    return EffectManager.instance;
  }
}
```

单例模式管理所有特效。

### 4.2 播放火焰特效

```typescript
public playFireEffect(position: THREE.Vector3, duration?: number): void {
  this.activeEffects.push(new FireEffect(position, duration === undefined ? Infinity : duration));
}
```

调用示例：
```typescript
EffectManager.getInstance().playFireEffect(new THREE.Vector3(0, 0, 0));
```

### 4.3 特效更新循环

```typescript
public update(delta: number): void {
  for (let i = this.activeEffects.length-1; i >= 0; i--) {
    const effect = this.activeEffects[i];
    effect.update(delta);
    if (!effect.isActive) {
      effect.dispose();
      this.activeEffects.splice(i, 1);
    }
  }
}
```

每帧调用 `update(delta)`，自动管理特效的生命周期。

### 4.4 调用时机

通常在游戏主循环中调用：
```typescript
function gameLoop(delta: number) {
  EffectManager.getInstance().update(delta);
  // 其他游戏逻辑...
}
```

## 五、数据流总结

```
FireEffect (3D粒子系统)
       │
       │ onUpdate 每帧调用
       ▼
┌─────────────────────────────────────┐
│  物理模拟：浮力、扩散、湍流、收缩     │
│  颜色变化：红白色 → 橙色 → 红色       │
│  位置更新：基于速度和阻力             │
└─────────────────────────────────────┘
       │
       │ 传递粒子数据（世界坐标 + 颜色）
       ▼
Flame2DEffect (2D轮廓渲染，单例)
       │
       │ render 每帧调用
       ▼
┌─────────────────────────────────────┐
│  粒子投影：3D → 屏幕2D坐标           │
│  包围盒计算                         │
│  模板选择：侧面 / 俯视（基于相机角度） │
│  颜色过滤：根据y位置选择红白/红色     │
│  模板缩放：适配粒子云大小            │
│  平滑曲线：Catmull-Rom样条          │
└─────────────────────────────────────┘
       │
       │ 绘制
       ▼
  Canvas 2D 轮廓 + 3D 圆形粒子
```

## 六、关键参数
