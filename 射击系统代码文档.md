#### 2.1.1 子弹方向计算函数

```typescript
// 实时计算子弹方向（纯函数，每次调用都用最新参数）
const getBulletDirection = (
  camera: THREE.Camera,
  characterPos: { x: number; y: number; z: number },
  mouseX: number,
  mouseY: number,
  canvasElement: HTMLCanvasElement  // 传入 canvas 元素
): THREE.Vector3 => {
  // 1. 计算 canvas 相对坐标
  const rect = canvasElement.getBoundingClientRect();
  const ndcX = ((mouseX - rect.left) / rect.width) * 2 - 1;
  const ndcY = -((mouseY - rect.top) / rect.height) * 2 + 1;

  // 2. 创建射线
  const raycaster = new THREE.Raycaster();
  raycaster.setFromCamera(new THREE.Vector2(ndcX, ndcY), camera);

  // 3. 直接使用相机射线方向作为子弹方向，考虑相机的所有旋转角度
  // 这样子弹会沿着玩家视线方向飞行，与相机俯仰角度保持一致
  return raycaster.ray.direction.clone().normalize();
};
```

**功能说明**：
- 计算鼠标在 canvas 上的相对坐标，并转换为标准化设备坐标 (NDC)
- 使用 THREE.Raycaster 创建从相机发出的射线，指向鼠标位置
- 直接使用射线的方向作为子弹方向，确保子弹沿着玩家视线飞行

// 鼠标事件处理
useEffect(() => {

### 2.2 子弹组件 (`src/systems/projectile/Bullet.tsx`)

子弹组件负责渲染子弹并处理其生命周期。

```typescript
import React, { useEffect } from 'react';
import { useFrame } from '@react-three/fiber';

interface BulletProps {
  position: { x: number; y: number; z: number };
  direction: { x: number; y: number; z: number };
  onExpire: (id: number) => void;
  id: number;
}

const Bullet: React.FC<BulletProps> = ({ position, direction, onExpire, id }) => {
  // 20秒后过期
  useEffect(() => {
    const timer = setTimeout(() => {
      onExpire(id);
    }, 20000);
    
    return () => clearTimeout(timer);
  }, [id, onExpire]);

  return (
    <mesh position={[position.x, position.y, position.z]}>
      <cylinderGeometry args={[0.05, 0.05, 0.2, 8]} />
      <meshStandardMaterial color="yellow" emissive="yellow" emissiveIntensity={0.5} />
    </mesh>
  );
};

export default Bullet;
```

**功能说明**：
- 渲染黄色子弹模型
- 设置20秒的过期计时器，过期后调用 `onExpire` 回调
- 使用圆柱体几何体创建子弹模型，设置黄色材质

## 3. 技术实现细节

### 3.1 相机控制与射击角度

```typescript
// 第三人称摄像机跟随和旋转
if (camera) {
  // 根据旋转角度计算摄像机位置
  // 考虑俯仰角度的影响，抬高相机角度
  const pitchFactor = Math.cos(cameraPitch);
  const cameraTargetX = finalPos.x + Math.sin(cameraYaw) * cameraDistance * pitchFactor;
  const cameraTargetZ = finalPos.z + Math.cos(cameraYaw) * cameraDistance * pitchFactor;
  const cameraTargetY = finalPos.y + cameraHeight + Math.sin(cameraPitch) * cameraDistance * 1.5; // 增加1.5倍的俯仰影响，抬高角度
  
  // 平滑移动摄像机到目标位置，增强修正效果
  camera.position.x += (cameraTargetX - camera.position.x) * 0.3; // 进一步增加平滑因子，让相机更快地移动到目标位置
  camera.position.y += (cameraTargetY - camera.position.y) * 0.3;
  camera.position.z += (cameraTargetZ - camera.position.z) * 0.3;
  
  // 让摄像机看向角色
  camera.lookAt(finalPos.x, finalPos.y + 1, finalPos.z);
}
```

**功能说明**：
- 计算相机的目标位置，考虑旋转角度和高度
- 平滑移动相机到目标位置，增强修正效果
- 让相机始终看向角色，确保视角稳定

## 4. 关键技术点

### 4.1 射线投射

使用 `THREE.Raycaster` 从相机发出射线，指向鼠标位置，计算子弹方向。这种方法确保子弹方向与玩家视线一致，提高射击准确性。

### 4.2 相机跟随

实现了第三人称相机跟随系统，包括：
- 鼠标滚轮调整相机距离
- 鼠标移动控制相机旋转
- 平滑相机移动，增强视觉效果
- 考虑俯仰角度，抬高相机视角

### 4.3 射击频率控制

使用时间戳和 `useRef` 控制射击频率，确保每 200 毫秒只发射一次子弹，避免连续发射导致性能问题。

### 4.4 子弹物理

实现了简单的子弹物理系统：
- 子弹沿直线飞行，速度固定
- 子弹有轻微的下坠效果（通过方向向量的 Y 分量）
- 子弹超出边界后自动移除
- 子弹 20 秒后过期，避免内存泄漏

## 5. 使用说明

1. **鼠标控制**：

   - 鼠标滚轮：调整相机距离

   - 子弹 20 秒后自动消失

## 6. 常见问题及解决方案

### 6.1 子弹方向不准确

**问题**：子弹没有沿着准星方向飞行。

**解决方案**：
- 确保使用最新的相机状态计算子弹方向
- 正确计算鼠标在 canvas 上的相对坐标
- 直接使用相机射线方向作为子弹方向

### 6.2 射击时子弹突然多出来

**问题**：连续射击时子弹数量异常增加。

**解决方案**：
- 使用 `lastFireTimeRef` 控制射击频率
- 确保事件监听器只添加一次
- 检查子弹更新逻辑，确保子弹正确移除

### 6.3 摄像机靠近角色时射击角度异常

**问题**：相机靠近角色后，射击角度变得不准确。

**解决方案**：
- 增强相机修正效果，让相机更快地移动到目标位置
- 调整子弹发射高度，确保子弹能够接触到准心
- 确保相机位置和旋转先于开火检测更新
