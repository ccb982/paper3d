## 3. 纹理样式

### 3.1 静态样式

```
+----------+
| RRRRRRRR |  <- 上半部分：红色 (#FF0000)
| RRRRRRRR |
|----------|  <- 分界线
| BBBBBBBB |  <- 下半部分：蓝色 (#0000FF)
| BBBBBBBB |
+----------+
```

- **上半部分（4行）**：纯红色 `#FF0000`
- **下半部分（4行）**：纯蓝色 `#0000FF`
- **分界线**：水平中线，无渐变过渡

### 3.2 流动效果

- **上半部分（红色）**：带波浪边界的流动
- **下半部分（蓝色）**：带波浪边界的流动
- **交界处**：动态正弦波形边界 + 随机杂波，随时间流动和变化
- **流动速度**：2.0单位/秒（更快的流动速度）
- **波浪振幅**：动态正弦变化，在0.5-1.5像素之间
- **波浪频率**：动态正弦变化，在0.2-1.0之间
- **随机杂波**：每帧生成的随机噪声成分，使波形更加不规则
- **流动方式**：正弦曲线控制的平滑波浪流动 + 随机杂波，波浪形状随时间复杂变化
- **循环方式**：无缝循环（当纹理流动到边缘时，会从另一侧重新出现）

## 4. 文件结构

### 4.1 TestRedBlueTexture.ts

**文件路径**：`src/systems/textures/TestRedBlueTexture.ts`

**核心代码结构**：

```typescript
import * as THREE from 'three';

interface ITextureGenerator {
  type: 'canvas' | 'shader';
  generate(): THREE.Texture | THREE.Material;
  update(delta?: number): void;
  dispose(): void;
}

export class TestRedBlueTexture implements ITextureGenerator {
  type: 'canvas' | 'shader' = 'canvas';
  private texture: THREE.Texture | null = null;

  private time: number = 0;

  generate(): THREE.Texture | THREE.Material {
    const canvas = document.createElement('canvas');
    canvas.width = 8;
    canvas.height = 8;
    
    const context = canvas.getContext('2d');
    if (!context) {
      throw new Error('无法创建canvas上下文');
    }

    // 流动效果参数
    const flowSpeed = 2.0; // 增加流动速度
    const flowOffset = this.time * flowSpeed;
    const waveAmplitude = 4; // 增加波浪幅度
    const boundaryAmplitude = 1.5; // 交界处波浪幅度

    // 绘制红色上半部分（带波浪边界）
    context.fillStyle = '#ff0000';
    context.beginPath();
    context.moveTo(0, 0);
    
    // 绘制波浪边界
    for (let x = 0; x <= 8; x++) {
      const y = 4 + Math.sin(x * 0.8 + flowOffset) * boundaryAmplitude;
      context.lineTo(x, y);
    }
    
    context.lineTo(8, 0);
    context.closePath();
    context.fill();
    
    // 绘制蓝色下半部分（带波浪边界）
    context.fillStyle = '#0000ff';
    context.beginPath();
    context.moveTo(0, 8);
    
    // 绘制波浪边界（与上半部分对称）
    for (let x = 0; x <= 8; x++) {
      const y = 4 + Math.sin(x * 0.8 + flowOffset) * boundaryAmplitude;
      context.lineTo(x, y);
    }
    
    context.lineTo(8, 8);
    context.closePath();
    context.fill();

    if (!this.texture) {
      this.texture = new THREE.CanvasTexture(canvas);
    } else {
      (this.texture as THREE.CanvasTexture).image = canvas;
    }
    this.texture.needsUpdate = true;
    return this.texture;
  }

  update(delta?: number): void {
    if (delta && this.texture) {
      this.time += delta * 0.5; // 控制流动速度
      this.generate(); // 重新生成纹理以实现流动效果
    }
  }

  dispose(): void {
    if (this.texture) {
      this.texture.dispose();
    }
  }
}
```

## 5. 使用方式

## 10. 注意事项

1. **纹理复用**：纹理在首次调用 `generate()` 后会被缓存，后续调用直接返回缓存的纹理对象。

2. **内存管理**：使用完毕后，调用 `dispose()` 方法可以释放纹理占用的内存。

3. **单例模式**：TextureManager 使用单例模式，全局只有一个实例。

4. **注册覆盖**：如果注册同名的纹理，旧纹理会被新纹理覆盖。

5. **引用计数**：TextureManager 内部使用引用计数管理纹理，当引用计数为0时会自动释放纹理。
