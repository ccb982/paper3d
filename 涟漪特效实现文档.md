# 涟漪特效实现文档

## 1. 特效概述

当前实现的涟漪特效是一种通过着色器（Shader）技术在地面上绘制动态环形波纹的视觉效果。该特效具有以下特点：

- **动态效果**：波纹会随时间自动旋转，呈现出流动感
- **多层结构**：包含3个不同半径的环形波纹
- **局部显示**：每个环只显示一段弧，而非完整的圆环
- **UV映射**：通过UV坐标将效果投影到地面上
- **不影响其他特效**：使用不透明材质，避免影响其他视觉元素

## 2. 实现原理

### 2.1 技术选型

- **着色器实现**：使用Three.js的ShaderMaterial
- **投影方式**：通过UV坐标映射到地面
- **动画控制**：通过uniform变量uTime控制时间流逝

### 2.2 核心算法

1. **坐标转换**：将UV坐标从[0,1]映射到[-0.5,0.5]，使中心为原点
2. **极坐标计算**：计算每个像素的半径(r)和角度(angle)
3. **环形区域判断**：只在特定半径范围内绘制波纹
4. **弧段判断**：每个环只显示随时间旋转的一段弧
5. **颜色混合**：将波纹颜色与地面纹理混合

## 3. 代码实现

### 3.1 材质定义

```typescript
const groundMaterial = new THREE.ShaderMaterial({
  uniforms: {
    uGroundTexture: { value: groundTexture },
    uTime: { value: 0 }
  },
  vertexShader: `
    varying vec2 vUv;
    void main() {
      vUv = uv;
      gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
    }
  `,
  fragmentShader: `
    uniform sampler2D uGroundTexture;
    uniform float uTime;
    varying vec2 vUv;
    
    void main() {
      // 地面纹理采样
      vec4 groundColor = texture2D(uGroundTexture, vUv * 10.0);
      
      // 将 UV 坐标从 [0,1] 映射到 [-0.5,0.5]，使中心为 (0,0)
      vec2 p = vUv - 0.5;
      float r = length(p);
      float angle = atan(p.y, p.x);
      
      // 只显示半径在 0.1 到 0.4 之间的环形区域
      float ripple = 0.0;
      if (r < 0.4 && r > 0.1) {
        // 定义多层环的参数
        int ringCount = 3;
        for (int i = 0; i < ringCount; i++) {
          float ringRadius = 0.15 + float(i) * 0.1;
          float ringWidth = 0.02;
          float radiusDiff = abs(r - ringRadius);
          if (radiusDiff < ringWidth) {
            // 环的弧段：起始角度随时间偏移
            float startAngle = uTime * 0.5 + float(i) * 1.2;
            float arcLength = 0.6;
            float endAngle = startAngle + arcLength;
            // 角度归一化
            float a = angle;
            if (a < 0.0) a += 6.28318;
            float start = mod(startAngle, 6.28318);
            float end = mod(endAngle, 6.28318);
            if ((start < end && a >= start && a <= end) || (start >= end && (a >= start || a <= end))) {
              ripple = 1.0;
              break;
            }
          }
        }
      }
      
      // 颜色：蓝白色
      vec3 rippleColor = vec3(0.3, 0.5, 0.9);
      float rippleAlpha = 0.8 * ripple;
      
      // 混合地面纹理和涟漪效果
      vec3 finalColor = mix(groundColor.rgb, rippleColor, rippleAlpha);
      gl_FragColor = vec4(finalColor, 1.0);
    }
  `,
  side: THREE.DoubleSide
});
```

### 3.2 动画更新

```typescript
// 在GameWorld.tsx的useFrame函数中
useFrame(({ camera }, delta) => {
  if (mode !== GameMode.BATTLE && mode !== GameMode.DAILY) {
    return;
  }
  
  // 更新特效管理器
  EffectManager.getInstance().update(delta);
  
  // 更新涟漪效果的时间
  if (terrainRef.current) {
    const terrain = terrainRef.current;
    if (terrain.material && 'uniforms' in terrain.material) {
      terrain.material.uniforms.uTime.value += delta;
    }
  }
});
```

## 4. 技术参数

| 参数 | 描述 | 当前值 | 调整建议 |
|------|------|--------|----------|
| 环形区域半径 | 波纹的整体大小 | 0.1-0.4 | 增大值可使波纹范围更大 |
| 环数量 | 波纹的层数 | 3 | 可根据需要增减 |
| 环间距 | 各层波纹之间的距离 | 0.1 | 增大值可使波纹更分散 |
| 环宽度 | 每层波纹的厚度 | 0.02 | 增大值可使波纹更粗 |
| 弧段长度 | 每个环显示的弧长 | 0.6 | 增大值可使弧段更长 |
| 旋转速度 | 波纹旋转的速度 | 0.5 | 增大值可使旋转更快 |
| 波纹颜色 | 波纹的颜色 | (0.3, 0.5, 0.9) | 可根据场景调整 |
| 波纹透明度 | 波纹的透明程度 | 0.8 | 增大值可使波纹更明显 |

## 6. 启用/禁用方法

### 6.1 禁用涟漪特效

要禁用涟漪特效，有两种方法：

**方法一：修改着色器代码**

在`TerrainRenderer.tsx`文件中，修改片元着色器，直接返回地面纹理颜色：

```glsl
void main() {
  // 地面纹理采样
  vec4 groundColor = texture2D(uGroundTexture, vUv * 10.0);
  
  // 直接返回地面颜色，不添加涟漪效果
  gl_FragColor = groundColor;
}
```

**方法二：切换材质**

将`ShaderMaterial`改回`MeshStandardMaterial`：

```typescript
const groundMaterial = new THREE.MeshStandardMaterial({
  map: groundTexture,
  side: THREE.DoubleSide
});
```

### 6.2 调整涟漪效果

要调整涟漪效果的参数，修改`TerrainRenderer.tsx`文件中的片元着色器：

- **调整大小**：修改半径判断条件 `if (r < 0.4 && r > 0.1)`
- **调整速度**：修改时间系数 `uTime * 0.5`
- **调整颜色**：修改 `vec3 rippleColor = vec3(0.3, 0.5, 0.9)`
- **调整透明度**：修改 `float rippleAlpha = 0.8 * ripple`

## 7. 性能考虑

- **计算复杂度**：每个像素都需要进行极坐标计算和多层环判断，但计算量相对较小
- **渲染性能**：使用着色器实现，GPU加速，性能表现良好
- **内存占用**：只增加了一个uniform变量，内存占用可忽略

## 8. 未来优化方向

1. **参数化控制**：将涟漪效果的参数（如大小、速度、颜色等）提取为uniform变量，便于外部控制
2. **特效管理器集成**：扩展特效管理器，使其能够控制材质级别的特效
3. **交互触发**：添加触发机制，如角色踩在地面上时产生涟漪
4. **噪声扰动**：添加噪声函数，使涟漪边缘更加自然
5. **多波纹支持**：支持多个独立的涟漪效果同时存在

## 9. 常见问题与解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 涟漪范围过大 | 半径范围设置过大 | 修改半径判断条件，减小最大值 |
| 涟漪旋转过快 | 时间系数设置过大 | 减小时间系数，如改为 `uTime * 0.2` |
| 涟漪颜色过亮 | 颜色值设置过高 | 减小颜色分量值，如改为 `vec3(0.2, 0.4, 0.8)` |
| 影响其他特效 | 材质透明属性导致 | 确保 `transparent` 属性为 `false`，alpha值为1.0 |

## 10. 总结

当前实现的涟漪特效通过着色器技术，在地面上绘制了动态的环形波纹效果。该特效具有以下优点：

- **视觉效果好**：动态旋转的波纹给场景增添了活力
- **性能高效**：使用GPU加速，计算量小
- **实现简洁**：代码结构清晰，易于理解和修改
- **不影响其他特效**：使用不透明材质，避免了渲染冲突

通过调整相关参数，可以轻松改变涟漪的大小、速度、颜色等属性，以适应不同的场景需求。
