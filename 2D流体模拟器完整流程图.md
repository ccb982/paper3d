```
│  │                   (GPU 并行计算引擎)                                 │   │
│  │                                                                    │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │  速度场（双缓冲）                                              │   │   │
│  │  │  velPrevVar ◄──────┐                                        │   │   │
│  │  │                    │ swap()                                 │   │   │
│  │  │  velNextVar ───────┘                                        │   │   │

│  │  └─────────────────────────────────────────────────────────────┘   │   │

│  │                              ▼                                      │   │
```

## 二、每帧完整计算流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              每帧 update(dt)                                │
│                                                                             │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 0 步】初始化                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  dt = min(传入的deltaTime, 0.033)  // 限制最大时间步长                │   │
│  │  resolution = [width, height]                                        │   │
│  │  dtReinit = 0.5 / min(width, height)                               │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 0.5 步】预计算固体法线（仅当 solidMask 改变时执行一次）               │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  solidMaskTexture ──[计算梯度]──→ solidNormalTexture                │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      solid_r = solidMask[x+1, y]                              │  │   │
│  │  │      solid_l = solidMask[x-1, y]                              │  │   │
│  │  │      solid_t = solidMask[x, y+1]                              │  │   │
│  │  │      solid_b = solidMask[x, y-1]                              │  │   │
│  │  │                                                               │  │   │
│  │  │      // 法线 = 梯度，归一化                                     │  │   │
│  │  │      nx = solid_r - solid_l                                    │  │   │
│  │  │      ny = solid_t - solid_b                                    │  │   │
│  │  │      normal = normalize([nx, ny])                              │  │   │
│  │  │                                                               │  │   │
│  │  │      solidNormal[x,y] = normal                                 │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  ✅ 预计算后，每帧直接读取 solidNormalTexture，无需重复计算！        │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 1 步】速度平流                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: velocityAdvectionShader()                                  │   │
│  │  输入: velPrevVar                                                   │   │
│  │  输出: velNextVar                                                   │   │
│  │                                                                        │   │
│  │  公式: vel_new = vel_old (反推法)                                    │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(velPrev, [x,y])                         │  │   │
│  │  │      step = vel * dt / resolution                            │  │   │
│  │  │      back_uv = uv - step                                     │  │   │
│  │  │      vel_new = texture2D(velPrev, back_uv)                    │  │   │
│  │  │      output = vel_new                                        │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  然后: swap(velPrevVar, velNextVar)                                 │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 2 步】外力计算（重力 + 粘性 + 表面张力）                               │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: externalForcesShader()                                     │   │
│  │  输入: velPrevVar, phiPrevVar                                       │   │
│  │  输出: forcedVelVar                                                 │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(velPrev, [x,y])                         │  │   │
│  │  │      phi = texture2D(phiPrev, [x,y])                         │  │   │
│  │  │                                                               │  │   │
│  │  │      // ① 重力（恒定）                                        │  │   │
│  │  │      vel.y += gravity * dt                                    │  │   │
│  │  │                                                               │  │   │
│  │  │      // ② 粘性扩散                                            │  │   │
│  │  │      vel_laplacian = ...                                      │  │   │
│  │  │      vel += (viscosity / density) * dt * vel_laplacian       │  │   │
│  │  │                                                               │  │   │
│  │  │      // ③ 表面张力（仅在界面附近 |phi| < eps）               │  │   │
│  │  │      if (abs(phi) < eps) {                                   │  │   │
│  │  │          kappa = 计算曲率(...)                                │  │   │
│  │  │          f_st = surfaceTension * kappa * normal              │  │   │
│  │  │          vel += (f_st / density) * dt                        │  │   │
│  │  │      }                                                         │  │   │
│  │  │                                                               │  │   │
│  │  │      output = vel                                              │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 2.5 步】墙碰撞反弹（使用预计算法线）⭐ 新增                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: wallCollisionShader()                                      │   │
│  │  输入: forcedVelVar, solidNormalTexture（预计算）                  │   │
│  │  输出: velAfterCollisionVar                                        │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      isSolid = solidMask[x,y]                                  │  │   │
│  │  │                                                               │  │   │
│  │  │      if (isSolid < 0.5) {                                     │  │   │
│  │  │          output = texture2D(forcedVel, [x,y])  // 直接通过    │  │   │
│  │  │      } else {                                                 │  │   │
│  │  │          // ========== 墙区域：反弹计算 ==========              │  │   │
│  │  │          normal = texture2D(solidNormal, [x,y])  // 预计算！ │  │   │
│  │  │          vel = texture2D(forcedVel, [x,y])                   │  │   │
│  │  │                                                               │  │   │
│  │  │          // 分解速度                                           │  │   │
│  │  │          vn = dot(vel, normal)  // 法向分量                 │  │   │
│  │  │          vt = vel - vn * normal  // 切向分量                 │  │   │
│  │  │                                                               │  │   │
│  │  │          // 法向反向 × 恢复系数 e                             │  │   │
│  │  │          vn_new = -vn * restitution  // e = 0.8               │  │   │
│  │  │                                                               │  │   │
│  │  │          // 切向 × 摩擦系数                                   │  │   │
│  │  │          vt_new = vt * friction  // μ = 0.95                │  │   │
│  │  │                                                               │  │   │
│  │  │          vel_new = vt_new + vn_new * normal                 │  │   │
│  │  │          output = vel_new                                    │  │   │
│  │  │      }                                                         │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  ✅ 关键：直接读取预计算的 solidNormalTexture，无需每帧计算梯度！   │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 3 步】计算散度                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: divergenceShader()                                         │   │
│  │  输入: velAfterCollisionVar ◄── 使用碰撞后的速度！                 │   │
│  │  输出: divergenceVar                                                │   │
│  │                                                                        │   │
│  │  公式: div = ∂vx/∂x + ∂vy/∂y                                        │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vx_R = texture2D(velAfterCollision, [x+1,y]).x          │  │   │
│  │  │      vx_L = texture2D(velAfterCollision, [x-1,y]).x          │  │   │
│  │  │      vy_T = texture2D(velAfterCollision, [x,y+1]).y          │  │   │
│  │  │      vy_B = texture2D(velAfterCollision, [x,y-1]).y          │  │   │
│  │  │      div = (vx_R - vx_L) / 2dx + (vy_T - vy_B) / 2dy        │  │   │
│  │  │      output = div                                             │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 4 步】压力求解（Jacobi 迭代）                                         │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: pressureJacobiShader()                                    │   │
│  │  输入: divergenceVar, pressureVarA/B (ping-pong)                    │   │
│  │  输出: pressureVarA 或 pressureVarB                                  │   │
│  │                                                                        │   │
│  │  公式: p_new = (p_L + p_R + p_D + p_U - div*h²/ρ/dt) / 4          │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for i = 0 to pressureIterations:                            │  │   │
│  │  │      for each pixel (x, y):                                   │  │   │
│  │  │          p_L = texture2D(pressure, [x-1,y])                   │  │   │
│  │  │          p_R = texture2D(pressure, [x+1,y])                   │  │   │
│  │  │          p_D = texture2D(pressure, [x,y-1])                   │  │   │
│  │  │          p_U = texture2D(pressure, [x,y+1])                   │  │   │
│  │  │          div = texture2D(divergence, [x,y])                   │  │   │
│  │  │          h = 1 / resolution.x                                 │  │   │
│  │  │          p_new = (p_L+p_R+p_D+p_U - density/dt*div*h*h) / 4   │  │   │
│  │  │                                                               │  │   │
│  │  │          if (i 是偶数) output → pressureVarB                  │  │   │
│  │  │          else            output → pressureVarA                │  │   │
│  │  │          swap(pressureVarA, pressureVarB)                     │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 5 步】速度修正（压力梯度减法）                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: velocityCorrectShader()                                    │   │
│  │  输入: forcedVelVar, pressureVarA/B                                  │   │
│  │  输出: velNextVar (覆盖)                                             │   │
│  │                                                                        │   │
│  │  公式: vel_corrected = vel - ∇p / ρ                                  │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(forcedVel, [x,y])                         │  │   │
│  │  │      p_L = texture2D(pressure, [x-1,y])                       │  │   │
│  │  │      p_R = texture2D(pressure, [x+1,y])                       │  │   │
│  │  │      p_D = texture2D(pressure, [x,y-1])                       │  │   │
│  │  │      p_U = texture2D(pressure, [x,y+1])                       │  │   │
│  │  │      vel.x -= (dt / density) * (p_R - p_L) / (2dx)           │  │   │
│  │  │      vel.y -= (dt / density) * (p_U - p_D) / (2dy)           │  │   │
│  │  │      output = vel                                              │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 6 步】Level Set 平流                                                  │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: levelSetAdvectionShader()                                 │   │
│  │  输入: velNextVar, phiPrevVar                                       │   │
│  │  输出: phiNextVar                                                   │   │
│  │                                                                        │   │
│  │  公式: phi_new = phi_old (反推法)                                    │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(velNext, [x,y])                         │  │   │
│  │  │      step = vel * dt / resolution                            │  │   │
│  │  │      back_uv = uv - step                                     │  │   │
│  │  │      phi_new = texture2D(phiPrev, back_uv)                    │  │   │
│  │  │      output = phi_new                                         │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  然后: swap(phiPrevVar, phiNextVar)                                  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 7 步】Level Set 重新初始化（保持界面锐利）                            │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: levelSetReinitShader()                                     │   │
│  │  输入: phiPrevVar                                                   │   │
│  │  输出: phiNextVar                                                   │   │
│  │                                                                        │   │
│  │  公式: phi_new = phi - dt_reinit * sign(phi) * (|∇phi| - 1)        │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for i = 0 to reinitIterations:                              │  │   │
│  │  │      for each pixel (x, y):                                   │  │   │
│  │  │          phi_0 = texture2D(phi, [x,y])                       │  │   │
│  │  │          phi_grad = 计算|∇phi|(...)                          │  │   │

│  │  │      swap(phiPrevVar, phiNextVar)                              │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 8 步】固体边界处理（可选）                                            │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  如果 solidMaskTexture 存在：                                        │   │
```

## 五、墙碰撞反弹机制（新增）

### 5.1 核心思想

```
┌─────────────────────────────────────────────────────────────┐
│  墙碰撞反弹 vs 旧版 vel=0：                                │
│                                                              │
│  旧版（粘住不动）：                                          │
│  if (isSolid) { vel = vec2(0); }  // ❌ 水粘在墙上        │
│                                                              │
│  新版（反弹）：                                              │
│  if (isSolid) { vel = reflect(vel, normal) * e; }  // ✅   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 5.2 预计算法线 vs 每帧计算

```
┌─────────────────────────────────────────────────────────────┐
│  预计算法线（推荐）：静态固体只算一次                         │
│                                                              │
│  初始化时：                                                 │
│  solidMask ──[计算梯度]──→ solidNormalTexture（存起来）     │
│                                                              │
│  每帧：                                                     │
│  solidNormal ──[直接读取]──→ wallCollisionShader           │
│                                                              │
│  性能：1 次计算 + N 次读取                                  │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  每帧计算法线：动态固体必须每帧算                            │
│                                                              │
│  每帧：                                                     │
│  solidMask ──[计算梯度]──→ normal ──→ 反弹计算             │
│                                                              │
│  性能：N 次计算                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 反弹物理公式

```
┌─────────────────────────────────────────────────────────────┐
│  速度分解：                                                  │
│                                                              │
│      vel                                                     │
│        ↓                                                     │
│       ╱                                                      │
│      ╱  normal                                              │
│     ╱  ╱                                                     │
│    ╱  ╱                                                      │
│   ●──╱←──── vt (切向分量，保持方向)                          │
│      ↑                                                       │
│      vn (法向分量：反向 × e)                                 │
│                                                              │
│  分解：                                                     │
│  vn = dot(vel, normal)          // 法向分量                 │
│  vt = vel - vn * normal         // 切向分量                 │
│                                                              │
│  反弹：                                                     │
│  vn_new = -vn × restitution     // 法向反向 × e            │
│  vt_new = vt × friction         // 切向衰减                 │
│                                                              │
│  合并：                                                     │
│  vel_new = vt_new + vn_new × normal                        │
└─────────────────────────────────────────────────────────────┘
```

### 5.4 参数说明

| 参数 | 符号 | 推荐值 | 说明 |
|------|------|--------|------|
| 恢复系数 | restitution (e) | 0.7-0.8 | 0=粘住，1=完全弹性 |
| 摩擦系数 | friction (μ) | 0.95 | 0=完全光滑，1=无摩擦 |

## 七、关键物理公式汇总

### 7.1 Navier-Stokes 方程（简化版）

```
∂u/∂t + (u·∇)u = -∇p/ρ + ν∇²u + g + f_st/ρ
∇·u = 0
```

### 6.2 各步骤公式

| 步骤 | 公式 | 含义 |
|------|------|------|
| **平流** | u_new = u_old ○ u_old | Semi-Lagrangian 反推 |
| **重力** | u.y += g * dt | 恒定加速度 |
| **粘性** | u += ν∇²u * dt | 速度扩散 |
| **散度** | div = ∂u_x/∂x + ∂u_y/∂y | 体积变化率 |
| **压力** | ∇²p = div * ρ/dt | Poisson 方程 |
| **修正** | u_new = u_old - ∇p/ρ | 不可压缩条件 |
| **表面张力** | f_st = σκδn | 曲率力 |

### 7.3 Level Set 公式

| 步骤 | 公式 | 含义 |
|------|------|------|
| **初始化** | phi = ±(radius - dist) | 有符号距离 |
| **平流** | phi_new = phi ○ u | 跟着速度移动 |
| **重初始化** | phi = phi - dt*sign(phi)(\|∇phi\|-1) | 保持距离函数性质 |

## 九、完整调用示例

```typescript
// 创建模拟器
const simulator = new FluidSimulator(renderer, {
    width: 512,
    height: 512,
    density: 1000,           // kg/m³
    viscosity: 0.001,        // Pa·s
    surfaceTension: 0.072,   // N/m
    gravity: 9.8,            // m/s²
    pressureIterations: 30,
    reinitIterations: 5,
    timeStep: 0.016,
    restitution: 0.8,        // 墙碰撞恢复系数
    friction: 0.95           // 墙碰撞摩擦系数
});

// 设置固体掩码（同时预计算法线）
simulator.setSolidMaskTexture(solidMaskTexture);

// 每帧调用
function animate() {
    const dt = clock.getDelta();
    simulator.update(dt);

    // 渲染
    const phiTexture = simulator.getLevelSetTexture();
    renderQuad.material.map = phiTexture;
    renderer.render(scene, camera);

    requestAnimationFrame(animate);
}
```

## 十、注意事项

1. **时间步长限制**：dt 不能太大，否则数值不稳定
2. **压力迭代次数**：次数越多越精确，但性能下降
3. **Level Set 重初始化**：防止 phi 失去距离函数性质
4. **边界条件**：固体掩码需要在计算前应用
5. **性能优化**：网格越大计算越慢，需权衡精度和性能
6. **墙碰撞法线**：静态固体只算一次，动态固体需每帧计算
