```
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │                      纹理缓冲系统                                    │   │
│  │                                                                    │   │
│  │  ┌─────────────────────────────────────────────────────────────┐   │   │
│  │  │  速度场（双缓冲）                                              │   │   │
│  │  │  velTexA ◄─── curVelTex ───► velTexB                         │   │   │

│  │  └─────────────────────────────────────────────────────────────┘   │   │
```

## 二、每帧完整计算流程

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              每帧 update(dt)                                │
│                                                                             │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 0 步】初始化                                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  dt = min(传入的deltaTime, 0.033)  // 限制最大时间步长                │   │
│  │  resolution = [width, height]                                        │   │
│  │  dtReinit = 0.5 / min(width, height)                               │   │
│  │                                                                        │   │
│  │  // 自定义流体形状初始化（可选）                                       │   │
│  │  if (params.initialLevelSet) {                                       │   │
│  │      copyTextureToTarget(params.initialLevelSet, phiTexA/B)         │   │
│  │  } else {                                                            │   │
│  │      renderFullscreen(initLevelSetShader(), phiTexA/B)              │   │
│  │  }                                                                   │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 0.5 步】预计算固体法线（仅当 solidMask 改变时执行一次）               │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  solidMaskTex ──[计算梯度]──→ solidNormalTex                        │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                               │  │   │
│  │  │      solid_r = solidMask[x+1, y]                      │  │   │
│  │  │      solid_l = solidMask[x-1, y]                      │  │   │
│  │  │      solid_t = solidMask[x, y+1]                      │  │   │
│  │  │      solid_b = solidMask[x, y-1]                      │  │   │
│  │  │                                                               │  │   │
│  │  │      // 法线 = 梯度，归一化                                     │  │   │
│  │  │      nx = solid_r - solid_l                                    │  │   │
│  │  │      ny = solid_t - solid_b                                    │  │   │
│  │  │      normal = normalize([nx, ny])                              │  │   │
│  │  │                                                               │  │   │
│  │  │      solidNormal[x,y] = normal                                 │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  ✅ 预计算后，每帧直接读取 solidNormalTex，无需重复计算！        │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 1 步】固体边界清理 #1（可选）                                          │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: solidBoundaryClearShader()                               │   │
│  │  输入: curVelTex, curPhiTex, solidMaskTex                         │   │
│  │  输出: velTexA, phiTexA                                           │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      isSolid = texture2D(solidMask, uv).r                    │  │   │
│  │  │      vel = texture2D(velInput, uv).rg                         │  │   │
│  │  │      phi = texture2D(phiInput, uv).r                          │  │   │
│  │  │      if (isSolid > 0.5) {                                     │  │   │
│  │  │          vel = vec2(0.0)                                       │  │   │
│  │  │          phi = -1.0                                           │  │   │
│  │  │      }                                                         │  │   │
│  │  │      gl_FragColor = vec4(vel, phi, 1.0)                       │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  更新: curVelTex = velTexA, curPhiTex = phiTexA                    │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 2 步】速度平流                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: velocityAdvectionShader()                                  │   │
│  │  输入: curVelTex                                                    │   │
│  │  输出: velTexB                                                     │   │
│  │                                                                        │   │
│  │  公式: vel_new = vel_old (反推法)                                    │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(velocity, uv).rg                         │  │   │
│  │  │      step = vel * dt / resolution                            │  │   │
│  │  │      back_uv = uv - step                                     │  │   │
│  │  │      vel_new = texture2D(velocity, back_uv).rg               │  │   │
│  │  │      gl_FragColor = vec4(vel_new, 0.0, 1.0)                  │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  更新: curVelTex = velTexB                                          │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 3 步】外力计算（重力 + 粘性 + 表面张力）                               │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: externalForcesShader()                                     │   │
│  │  输入: curVelTex, curPhiTex                                         │   │
│  │  输出: forcedVelTex                                                 │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(velocity, uv).rg                         │  │   │
│  │  │      phi = texture2D(levelset, uv).r                          │  │   │
│  │  │                                                               │  │   │
│  │  │      // ① 重力（恒定）                                        │  │   │
│  │  │      vel.y += gravity * dt                                    │  │   │
│  │  │                                                               │  │   │
│  │  │      // ② 粘性扩散                                            │  │   │
│  │  │      laplacian = (vel_r + vel_l + vel_t + vel_b - 4*vel)      │  │   │
│  │  │      vel += (viscosity/density) * dt * laplacian              │  │   │
│  │  │                                                               │  │   │
│  │  │      // ③ 表面张力（仅在界面附近 |phi| < eps）               │  │   │
│  │  │      if (abs(phi) < eps) {                                   │  │   │
│  │  │          kappa = 计算曲率(...)                                │  │   │
│  │  │          f_st = surfaceTension * kappa * normal              │  │   │
│  │  │          vel += (f_st / density) * dt                        │  │   │
│  │  │      }                                                         │  │   │
│  │  │                                                               │  │   │
│  │  │      // ④ 流体注入（可选）                                    │  │   │
│  │  │      if (injectionEnabled) {                                  │  │   │
│  │  │          // 更新 vel 和 phi（存储在 B 通道）                   │  │   │
│  │  │      }                                                         │  │   │
│  │  │                                                               │  │   │
│  │  │      gl_FragColor = vec4(vel, phi, 1.0)                       │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 4 步】墙碰撞处理（使用预计算法线）⭐ 新增                              │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: wallCollisionShader()                                      │   │
│  │  输入: forcedVelTex, solidMaskTex, solidNormalTex（预计算）        │   │
│  │  输出: velAfterCollisionTex                                        │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      isSolid = texture2D(solidMask, uv).r                    │  │   │
│  │  │                                                               │  │   │
│  │  │      if (isSolid < 0.5) {                                     │  │   │
│  │  │          output = texture2D(forcedVel, uv).rg  // 直接通过    │  │   │
│  │  │      } else {                                                 │  │   │
│  │  │          // ========== 墙区域：反弹计算 ==========              │  │   │
│  │  │          normal = texture2D(solidNormal, uv).rg  // 预计算！ │  │   │
│  │  │          vel = texture2D(forcedVel, uv).rg                   │  │   │
│  │  │                                                               │  │   │
│  │  │          // 分解速度                                           │  │   │
│  │  │          vn = dot(vel, normal)  // 法向分量                 │  │   │
│  │  │          vt = vel - vn * normal  // 切向分量                 │  │   │
│  │  │                                                               │  │   │
│  │  │          // 法向反向 × 恢复系数 e                             │  │   │
│  │  │          vn_new = -vn * restitution  // e = 0.8               │  │   │
│  │  │                                                               │  │   │
│  │  │          // 切向 × 摩擦系数                                   │  │   │
│  │  │          vt_new = vt * friction  // μ = 0.95                │  │   │
│  │  │                                                               │  │   │
│  │  │          vel_new = vt_new + vn_new * normal                 │  │   │
│  │  │          gl_FragColor = vec4(vel_new, 0.0, 1.0)             │  │   │
│  │  │      }                                                         │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  ✅ 关键：直接读取预计算的 solidNormalTex，无需每帧计算梯度！   │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 5 步】计算散度                                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: divergenceShader()                                         │   │
│  │  输入: velAfterCollisionTex (或 forcedVelTex 若无固体)            │   │
│  │  输出: divergenceTex                                                │   │
│  │                                                                        │   │
│  │  公式: div = ∂vx/∂x + ∂vy/∂y                                        │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vx_R = texture2D(velocity, uv+dx).x                     │  │   │
│  │  │      vx_L = texture2D(velocity, uv-dx).x                     │  │   │
│  │  │      vy_T = texture2D(velocity, uv+dy).y                     │  │   │
│  │  │      vy_B = texture2D(velocity, uv-dy).y                     │  │   │
│  │  │      div = (vx_R - vx_L) / 2dx + (vy_T - vy_B) / 2dy        │  │   │
│  │  │      gl_FragColor = vec4(div, 0.0, 0.0, 1.0)                 │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 6 步】压力求解（Jacobi 迭代）                                         │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: pressureJacobiShader()                                    │   │
│  │  输入: divergenceTex, pressureTexA/B (ping-pong)                    │   │
│  │  输出: pressureTexB/A (交替)                                        │   │
│  │                                                                        │   │
│  │  公式: p_new = (p_L + p_R + p_D + p_U - div*h²/ρ/dt) / 4          │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for i = 0 to pressureIterations:                            │  │   │
│  │  │      for each pixel (x, y):                                   │  │   │
│  │  │          p_L = texture2D(pressure, uv-dx).r                   │  │   │
│  │  │          p_R = texture2D(pressure, uv+dx).r                   │  │   │
│  │  │          p_D = texture2D(pressure, uv-dy).r                   │  │   │
│  │  │          p_U = texture2D(pressure, uv+dy).r                   │  │   │
│  │  │          div = texture2D(divergence, uv).r                    │  │   │
│  │  │          h = 1 / resolution.x                                 │  │   │
│  │  │          p_new = (p_L+p_R+p_D+p_U - density/dt*div*h*h) / 4   │  │   │
│  │  │                                                               │  │   │
│  │  │          render → pressureDst                                 │  │   │
│  │  │          [pressureSrc, pressureDst] = swap()                  │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
│  │  更新: curPressureTex = pressureSrc                                │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                    │                                        │
│                                    ▼                                        │
│  ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│  【第 7 步】速度修正（压力梯度减法）                                        │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  Shader: velocityCorrectShader()                                    │   │
│  │  输入: velAfterCollisionTex, curPressureTex                        │   │
│  │  输出: velCorrectTex                                               │   │
│  │                                                                        │   │
│  │  公式: vel_corrected = vel - ∇p / ρ                                  │   │
│  │                                                                        │   │
│  │  ┌───────────────────────────────────────────────────────────────┐  │   │
│  │  │  for each pixel (x, y):                                       │  │   │
│  │  │      vel = texture2D(velocity, uv).rg                         │  │   │
│  │  │      p_L = texture2D(pressure, uv-dx).r                       │  │   │
│  │  │      p_R = texture2D(pressure, uv+dx).r                       │  │   │
│  │  │      p_D = texture2D(pressure, uv-dy).r                       │  │   │
│  │  │      p_U = texture2D(pressure, uv+dy).r                       │  │   │
│  │  │      vel.x -= (dt/density) * (p_R-p_L) / (2dx)               │  │   │
│  │  │      vel.y -= (dt/density) * (p_U-p_D) / (2dy)               │  │   │
│  │  │      gl_FragColor = vec4(vel, 0.0, 1.0)                       │  │   │
│  │  └───────────────────────────────────────────────────────────────┘  │   │
│  │                                                                        │   │
```

## 三、架构改进说明

### 1. 移除 GPUComputationRenderer 依赖

| 旧架构 | 新架构 |
|--------|--------|
| 使用 `GPUComputationRenderer` 管理变量 | 直接使用 `WebGLRenderTarget` |
| 通过 `Variable` 对象管理着色器和纹理 | 着色器返回完整 `ShaderMaterial` |
| 复杂的依赖关系配置 | 简单的纹理引用管理 |
| 存在 uniform 缓存冲突问题 | 每次创建新材质，避免缓存 |

### 2. 简化的纹理管理

```
旧架构：
velPrevVar ←→ velNextVar (通过 swap() 交换引用)

新架构：
curVelTex → velTexA 或 velTexB (直接赋值切换)
```

### 3. 统一渲染流程

```typescript
// 通用渲染函数
renderFullscreen(material, inputTextures, outputTarget, clear)
```

- 自动管理渲染目标切换
- 统一的 uniform 设置方式
- 着色器使用后立即 dispose

### 4. 性能优化

| 优化项 | 说明 |
|--------|------|
| 场景/相机复用 | 构造时创建，全程复用 |
| 着色器 dispose | 每次使用后立即释放 |
| 固体法线预计算 | 仅在 solidMask 改变时计算一次 |
| 纹理引用切换 | 直接赋值，无额外开销 |

### 5. 关键接口对照表

| 方法 | 功能 |
|------|------|
| `getLevelSetTexture()` | 获取当前 Level Set 纹理 |
| `getVelocityTexture()` | 获取当前速度场纹理 |
| `setSolidMaskTexture()` | 设置固体掩码（自动计算法线） |
| `setLevelSetTexture()` | 设置自定义 Level Set 形状 |
| `setInjectionEnabled()` | 启用/禁用流体注入 |
| `setInjectionPosition()` | 设置注入位置 |
| `setInjectionFlowRate()` | 设置注入流量 |
| `setInjectionVelocity()` | 设置注入速度 |
| `setInjectionSize()` | 设置注入区域大小 |
| `dispose()` | 释放所有资源 |
