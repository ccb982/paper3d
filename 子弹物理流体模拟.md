# 子弹物理流体模拟 - 技术文档

## 一、核心设计目标

| 目标 | 说明 |
|-----|------|
| **无激波/马赫环** | 完全移除前方激波效果（马赫环属于枪口） |
| **表层切向受力** | 仅在椭圆环区域施加切向速度，内部不受力 |
| **侧视卡门涡街** | 呈现为上下交替摆动的波浪，而非圆形漩涡 |
| **动态跟随** | 子弹在纹理中相对静止，尾焰向后喷射 |

## 二、实现原理

### 2.1 坐标系设定

- **画布尺寸**: 768 × 512 像素（延长宽度以容纳尾焰）
- **子弹位置**: 固定在右侧 (BULLET_X = 688, BULLET_Y = 256)
- **喷射方向**: 向后（负X方向），带轻微上下分量

### 2.2 核心参数

```typescript
// 椭圆参数（子弹侧面轮廓）
a = 30;      // X轴半径
b = 18;      // Y轴半径

// 喷射参数
nozzleDist = 15;    // 喷射口距离弹头中心
nozzleSpeed = 180;  // 喷射速度（强力）

// 流体模拟参数
curl = 4;                    // 低涡旋，避免回流
velocityDissipation = 0.15;  // 速度消耗
dyeDissipation = 0.18;       // 颜色消耗
```

### 2.3 注入逻辑

#### 阶段一：弹头高压区
```typescript
// 实心椭圆，速度为零（阻挡流体向前）
this.fluidDynamics.setVelocity(BULLET_X, BULLET_Y, 0, a, b, 0, 0);
this.fluidDynamics.setDye(BULLET_X, BULLET_Y, 0, a*0.85, b*0.75, [1.0, 0.95, 0.85]);
```

#### 阶段二：强力喷射口
```typescript
// 仅从椭圆后方半圈喷射（角度范围：0.5π ~ 1.5π）
const angles = [Math.PI * 0.5, Math.PI * 0.7, Math.PI, Math.PI * 1.3, Math.PI * 1.5];

// 纯向后喷射，带轻微上下分量
const vx = -nozzleSpeed * pulse * 1.2;
const vy = Math.sin(ang) * nozzleSpeed * 0.3 * pulse;
```

#### 阶段三：后方加速区
```typescript
// 在喷射口后方持续施加向后力，确保流体不回流
for (let dist = 25; dist <= 140; dist += 15) {
    const speed = nozzleSpeed * (1 - dist / 160) * 0.7;
    this.fluidDynamics.setVelocity(backX, BULLET_Y, 0, spread, spread*0.5, -speed, 0);
}
```

#### 阶段四：侧视卡门涡街（波浪扰动）
```typescript
// 上下边缘交替波动，模拟侧视效果
for (let dist = 40; dist <= 120; dist += 20) {
    const wavePhase = Math.sin(time*2.5 + dist*0.07);
    
    // 上边缘青色波动
    this.fluidDynamics.setVelocity(backX, BULLET_Y - spread + wavePhase*waveAmp, ...);
    
    // 下边缘粉色反向波动

```

#### 阶段五：湍流扰动

## 三、侧视卡门涡街效果说明

### 3.1 视角差异

| 视角 | 涡旋旋转轴 | 视觉表现 |
|-----|-----------|---------|
| 俯视 | 垂直于水平面 | 完整的圆形/椭圆形涡旋对 |
| 侧视 | 平行于水平面（垂直屏幕） | 上下交替摆动的波浪条纹 |

### 3.2 侧视实现要点

1. **无圆形漩涡**：从侧面看不到完整圆圈，因为旋转轴指向屏幕外
2. **交替彩色凸起**：上边缘青色，下边缘粉色，交替向后移动
3. **波浪形尾迹**：整体呈现蛇形扭动效果

## 五、运行效果

### 5.1 预期效果

1. **弹头**：白色椭圆，位于纹理右侧，固定不动
2. **喷射口**：蓝色强力喷射，带轻微脉动
3. **尾焰**：蓝色主体，向后延伸，逐渐消散
4. **波浪边缘**：青色（上）和粉色（下）交替波动
5. **无回流**：流体严格向后流动，不会向前返回

### 5.2 可调参数说明

| 参数 | 调整方向 | 效果 |
|-----|---------|------|
| `nozzleSpeed` | ↑ | 喷射力更强，尾焰更长 |
| `velocityDissipation` | ↑ | 尾焰消散更快，寿命更短 |
| `dyeDissipation` | ↑ | 颜色消失更快 |
| `curl` | ↑ | 涡旋增强（可能导致回流） |

## 七、注意事项

1. **避免回流**：`curl` 参数不宜过大，否则会产生向前的旋转分量
2. **性能优化**：画布尺寸不宜过大，建议保持在 1024x1024 以内
3. **纹理更新**：每帧调用 `texture.needsUpdate = true` 确保纹理刷新
