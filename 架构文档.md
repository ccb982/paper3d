# 项目架构文档

## 一、项目概述

本项目是一个基于 React + Three.js 的 3D 纸片风格游戏（Paper-like），核心玩法包含**日常行动**和**战斗**两种模式，支持**锁定式**和**自由式**两种射击系统。

### 1.1 核心特性

- **纸片风格渲染**：角色和敌人以 2.5D 纸片形式呈现
- **双射击系统**：锁定射击（自动瞄准）+ 自由射击（传统 FPS 风格）
- **页面管理器**：通过 `gameStore.mode` 控制页面切换
- **调试模式**：可视化射线、坐标信息、射击方向等调试信息

### 1.2 技术栈

| 技术 | 用途 |
|------|------|
| React 18 | 前端框架 |
| Three.js | 3D 渲染核心 |
| @react-three/fiber | React Three.js 绑定 |
| @react-three/drei | React 专用 3D 工具库 |
| Zustand | 轻量级全局状态管理 |
| Vite | 开发服务器和构建工具 |

### 4.2 角色位置监听器

**实现方式**：使用独立的 `CharacterPositionStore` 单例存储角色位置，通过节流更新减少 UI 重渲染。

```typescript
// src/systems/character/CharacterPositionStore.ts
export class CharacterPositionStore {
  private static instance: CharacterPositionStore;

  public position: THREE.Vector3 = new THREE.Vector3(0, 0, 0);
  public velocity: THREE.Vector3 = new THREE.Vector3(0, 0, 0);
  public isMoving: boolean = false;

  // 单例模式实现
  public static getInstance(): CharacterPositionStore {
    if (!CharacterPositionStore.instance) {
      CharacterPositionStore.instance = new CharacterPositionStore();
    }
    return CharacterPositionStore.instance;
  }

  // 设置位置
  public setPosition(x: number, y: number, z: number): void {
    this.position.set(x, y, z);
  }

  // 获取位置副本（用于 UI 显示）
  public getPositionCopy(): { x: number; y: number; z: number } {
    return {
      x: this.position.x,
      y: this.position.y,
      z: this.position.z
    };
  }
}

export const characterPositionStore = CharacterPositionStore.getInstance();
```

**UI 更新节流**：

```typescript
// App.tsx - 每 100ms 更新一次 UI 显示
useEffect(() => {
  let animationId: number;
  const throttledUpdate = (timestamp: number) => {
    if (timestamp - lastUpdateRef.current >= 100) {
      setDisplayPos(characterPositionStore.getPositionCopy());
      lastUpdateRef.current = timestamp;
    }
    animationId = requestAnimationFrame(throttledUpdate);
  };
  animationId = requestAnimationFrame(throttledUpdate);
  return () => {
    cancelAnimationFrame(animationId);
  };
}, []);
```

### 5.3 锁定式射击 (LockonShootingSystem)

#### 核心机制

1. **多射线检测**：使用 5 条射线检测目标
   - 中心射线 + 上下左右 4 条偏移射线
   - 俯角越大，射线间距越大（适应不同视角）

2. **锁定流程**：
   ```
   鼠标按下 → 开始锁定 → 检测目标 → 显示锁定进度 → 1秒后锁定成功 → 自动射击
   ```

3. **锁定状态**：
   - `isLocking: false` - 未在锁定
   - `isLocking: true, lockCountdown > 0` - 锁定中（显示进度圆圈）
   - `lockedTarget !== null` - 锁定成功

#### 关键代码逻辑

```typescript
// 锁定式射击核心逻辑
private performRaycastAndLock(delta: number): void {
  // 1. 射线检测
  for (const offset of rayOffsets) {
    raycaster.setFromCamera(new THREE.Vector2(ndc.x + offset.x, ndc.y + offset.y), camera);
    const intersects = raycaster.intersectObjects(this.shootableObjects, true);
    allIntersects.push(...intersects);
  }

  // 2. 检测目标
  if (hasTarget && this.isMouseDown && !this.lockedTarget) {
    this.isLocking = true;
    this.lockCountdown = this.lockDuration; // 1000ms
    this.notifyLockStateChanged(true, this.lockCountdown);
  }

  // 3. 锁定计时
  if (this.isLocking) {
    this.lockCountdown -= delta * 1000;
    if (this.lockCountdown <= 0) {
      this.isLocking = false;
      console.log('锁定完成，可以射击');
    }
  }

  // 4. 锁定成功后自动射击
  if (this.lockedTarget && !this.isLocking && this.isMouseDown) {
    this.createBullet(position, direction);
  }
}
```

### 5.4 自由式射击 (FreeStyleShootingSystem)

#### 核心机制

1. **单射线检测**：从相机发射一条射线，检测前方目标
2. **指针锁定**：需要 `requestPointerLock()` 隐藏鼠标
3. **点击发射**：鼠标按下时持续发射子弹

#### 关键代码逻辑

```typescript
// 自由式射击核心逻辑
private updateShooting(): void {
  if (!this.isMouseDown) return;

  // 获取射线命中点
  const hitPoint = this.getRaycastHitPoint();
  if (!hitPoint) return;

  // 计算射击方向
  const direction = hitPoint.clone().sub(bulletOrigin).normalize();

  // 创建子弹
  this.createBullet(bulletOrigin, direction);
}
```

### 5.5 射击模式行为对比

| 特性 | 锁定式射击 | 自由式射击 |
|------|-----------|-----------|
| **射线数量** | 5 条 | 1 条 |
| **指针锁定** | ❌ 不需要 | ✅ 需要 |
| **鼠标显示** | 显示 | 隐藏 |
| **射击触发** | 锁定成功后自动 | 鼠标按下时 |
| **目标选择** | 自动选择最近目标 | 射线检测方向 |
| **适用场景** | 日常模式（强制） | 战斗模式（可选） |

### 5.6 鼠标锁定控制

```typescript
// GameWorld.tsx - 模式切换时的鼠标锁定处理
useEffect(() => {
  const gameContainer = document.querySelector('.game-container');
  const needsPointerLock = (mode === GameMode.BATTLE && activeShootingSystem === 'freestyle');

  if (needsPointerLock) {
    // 请求指针锁定
    gameContainer.requestPointerLock();
    gameContainer.style.cursor = 'none';
  } else {
    // 释放指针锁定
    document.exitPointerLock();
    gameContainer.style.cursor = 'default';
  }
}, [mode, activeShootingSystem]);

// 模式切换时的射击系统状态重置
useEffect(() => {
  if (mode === GameMode.DAILY) {
    setActiveShootingSystem('lockon');
    if (shootingManager) {
      shootingManager.setActiveSystem('lockon');
      onActiveSystemChanged('lockon');
    }
  } else if (mode !== GameMode.BATTLE) {
    // 在非战斗和非日常模式下，重置射击系统状态
    setActiveShootingSystem('lockon');
    if (shootingManager) {
      shootingManager.setActiveSystem('lockon');
      onActiveSystemChanged('lockon');
    }
  }
}, [mode, shootingManager, onActiveSystemChanged]);
```

### 9.1 实现

```typescript
// src/core/eventBus.ts
class EventBus {
  private events: Map<string, EventCallback[]> = new Map();

  on(event: string, callback: EventCallback): void {
    if (!this.events.has(event)) this.events.set(event, []);
    this.events.get(event)!.push(callback);
  }

  off(event: string, callback: EventCallback): void {
    const callbacks = this.events.get(event);
    if (callbacks) {
      const index = callbacks.indexOf(callback);
      if (index !== -1) callbacks.splice(index, 1);
    }
  }

  emit(event: string, ...args: any[]): void {
    const callbacks = this.events.get(event);
    if (callbacks) callbacks.forEach(cb => cb(...args));
  }
}

export const eventBus = new EventBus();
```

   ```typescript
  }
   ```

```typescript
    if (this.scene) {
      this.scene.add(entity.mesh);
    }
  }

  public removeEntity(entity: Entity): void {
    this.entities.delete(entity.id);
    if (this.scene && entity.mesh.parent) {
      this.scene.remove(entity.mesh);
    }
    entity.onDestroy();
  }

  public getEntitiesByType(type: string): Entity[] {
    return Array.from(this.entities.values()).filter(e => e.type === type);
  }

  public getAllEntities(): Entity[] {
    return Array.from(this.entities.values());
  }

  public update(delta: number): void {
    const toRemove: Entity[] = [];
    for (const entity of this.entities.values()) {
      if (entity.isActive) {
        entity.update(delta);
      } else {
        toRemove.push(entity);
      }
    }
    toRemove.forEach(entity => this.removeEntity(entity));
  }

  public clear(): void {
    for (const entity of this.entities.values()) {
      if (this.scene && entity.mesh.parent) {
        this.scene.remove(entity.mesh);
      }
    }
    this.entities.clear();
  }
}
```

### 15.6 碰撞管理器

```typescript
// src/core/CollisionManager.ts
class GridSpatialPartition {
  private cellSize: number;
  private grid: Map<string, Entity[]>;

  constructor(cellSize: number = 5) {
    this.cellSize = cellSize;
    this.grid = new Map();
  }

  public rebuild(entities: Entity[]): void {
    this.grid.clear();
    for (const entity of entities) {
      const key = this.getCellKey(entity.position);
      if (!this.grid.has(key)) {
        this.grid.set(key, []);
      }
      this.grid.get(key)!.push(entity);
    }
  }

  private getCellKey(position: THREE.Vector3): string {
    const x = Math.floor(position.x / this.cellSize);
    const z = Math.floor(position.z / this.cellSize);
    return `${x},${z}`;
  }

  public getPotentialCollisions(entity: Entity): Entity[] {
    const key = this.getCellKey(entity.position);
    const collisions: Entity[] = [];
    const checked = new Set<string>();

    for (let dx = -1; dx <= 1; dx++) {
      for (let dz = -1; dz <= 1; dz++) {
        const neighborKey = `${parseInt(key.split(',')[0]) + dx},${parseInt(key.split(',')[1]) + dz}`;
        const cellEntities = this.grid.get(neighborKey);
        if (cellEntities) {
          for (const other of cellEntities) {
            if (other.id !== entity.id && !checked.has(other.id)) {
              collisions.push(other);
              checked.add(other.id);
            }
          }
        }
      }
    }
    return collisions;
  }
}

type CollisionCallback = (a: Entity, b: Entity) => void;

export class CollisionManager {
  private static instance: CollisionManager;
  private collisionCallbacks: Map<string, CollisionCallback[]> = new Map();
  private spatialPartition: GridSpatialPartition;
  private collisionCooldowns: Map<string, number> = new Map();

  private constructor() {
    this.spatialPartition = new GridSpatialPartition();
  }

  public static getInstance(): CollisionManager {
    if (!CollisionManager.instance) {
      CollisionManager.instance = new CollisionManager();
    }
    return CollisionManager.instance;
  }

  public registerCollision(typeA: string, typeB: string, callback: CollisionCallback): void {
    const key = `${typeA}:${typeB}`;
    if (!this.collisionCallbacks.has(key)) {
      this.collisionCallbacks.set(key, []);
    }
    this.collisionCallbacks.get(key)!.push(callback);
  }

  public update(entities: Entity[]): void {
    this.spatialPartition.rebuild(entities);

    for (const entity of entities) {
      const potentialCollisions = this.spatialPartition.getPotentialCollisions(entity);
      for (const other of potentialCollisions) {
        if (this.canCollide(entity, other)) {
          this.checkCollision(entity, other);
        }
      }
    }
  }

  private canCollide(a: Entity, b: Entity): boolean {
    const key = this.getCollisionKey(a, b);
    const lastCollision = this.collisionCooldowns.get(key);
    const now = Date.now();

    if (lastCollision && now - lastCollision < 100) {
      return false;
    }

    this.collisionCooldowns.set(key, now);
    return true;
  }

  private getCollisionKey(a: Entity, b: Entity): string {
    return a.id < b.id ? `${a.id}:${b.id}` : `${b.id}:${a.id}`;
  }

  private checkCollision(a: Entity, b: Entity): void {
    const distance = a.position.distanceTo(b.position);
    if (distance < a.radius + b.radius) {
      const key1 = `${a.type}:${b.type}`;
      const key2 = `${b.type}:${a.type}`;

      if (this.collisionCallbacks.has(key1)) {
        this.collisionCallbacks.get(key1)!.forEach(callback => callback(a, b));
      }

      if (this.collisionCallbacks.has(key2)) {
        this.collisionCallbacks.get(key2)!.forEach(callback => callback(b, a));
      }
    }
  }
}
```

```typescript
      map: new THREE.TextureLoader().load(texturePath),
      side: THREE.DoubleSide 
    });
    const mesh = new THREE.Mesh(geometry, material);
    
    super(id, 'enemy', mesh);
    this.faction = 'enemy';
    this.position.copy(position);
    this.mesh.position.copy(position);
    this.radius = 0.5;
    this.health = 30;
    this.maxHealth = 30;
    this.speed = 3;
  }

  public update(delta: number): void {
    this.updateAI(delta);
    this.updatePosition(delta);
  }

  protected updateAI(delta: number): void {
    const mode = useGameStore.getState().mode;

    if (mode !== GameMode.BATTLE) {
      this.updateRandomMovement(delta);
    } else {
      this.updateChaseMovement(delta);
    }
  }

  private updateRandomMovement(delta: number): void {
    this.directionChangeTimer += delta * 1000;
    if (this.directionChangeTimer >= this.directionChangeInterval) {
      this.randomDirection.set(
        (Math.random() - 0.5) * 2,
        0,
        (Math.random() - 0.5) * 2
      ).normalize();
      this.directionChangeTimer = 0;
      this.directionChangeInterval = 2000 + Math.random() * 3000;
    }
    this.moveDirection.copy(this.randomDirection);
  }

  private updateChaseMovement(delta: number): void {
    const target = EntityManager.getInstance().getEntitiesByType('friendly')
      .find(e => (e as CharacterEntity).isPlayerControlled === true) as CharacterEntity;
    if (!target) return;
    
    const direction = new THREE.Vector3().subVectors(target.position, this.position);
    direction.y = 0;
    if (direction.length() > 0.01) {
      this.moveDirection.copy(direction).normalize();
    } else {
      this.moveDirection.set(0, 0, 0);
    }
  }

  private updatePosition(delta: number): void {
    if (this.moveDirection.length() > 0) {
      const moveVector = this.moveDirection.clone().multiplyScalar(this.speed * delta);
      this.position.add(moveVector);
      this.mesh.position.copy(this.position);
    }
  }

  protected onDeath(): void {
    super.onDeath();
    eventBus.emit('entity:death', { id: this.id, faction: this.faction });
  }
}
```

```typescript
// src/entities/BulletEntity.ts
export class BulletEntity extends Entity {
  private velocity: THREE.Vector3;
  private damage: number;
  private lifetime: number = 5;
  private age: number = 0;

  constructor(
    position: THREE.Vector3,
    direction: THREE.Vector3,
    velocity: number,
    color: number = 0xffaa00,
    damage: number = 10
  ) {
    const geometry = new THREE.SphereGeometry(0.1, 8, 8);
    const material = new THREE.MeshStandardMaterial({ color });
    const mesh = new THREE.Mesh(geometry, material);
    
    super(`bullet-${Date.now()}-${Math.random()}`, 'bullet', mesh);
    
    this.position.copy(position);
    this.mesh.position.copy(position);
    
    this.velocity = direction.clone().normalize().multiplyScalar(velocity);
    this.damage = damage;
    this.radius = 0.1;
  }

  public update(delta: number): void {
    if (!this.isActive) return;
    
    this.position.add(this.velocity.clone().multiplyScalar(delta));
    this.mesh.position.copy(this.position);
    
    this.age += delta;
    if (this.age >= this.lifetime) {
      this.isActive = false;
    }
  }

  public getDamage(): number {
    return this.damage;
  }

  protected onHit(entity: Entity): void {
    if (typeof (entity as any).takeDamage === 'function') {
      (entity as any).takeDamage({
        amount: this.damage,
        type: 'physical',
        source: this
      });
    }
    this.isActive = false;
  }
}
```
