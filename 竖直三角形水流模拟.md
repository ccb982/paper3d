# 竖直三角形水流模拟 - 技术文档

## 概述

`VerticalTriangleTexture` 是一个基于流体动力学的水流模拟纹理生成器，模拟水滴垂直下落并撞击三角形固体障碍物的物理过程。该模块使用 `@bienehito/fluid-dynamics` 流体动力学库实现实时流体渲染。

## 核心组件详解

### 1. 构造函数 (constructor)

```typescript
constructor() {
  // 1. 创建画布
  this.canvas = document.createElement('canvas');
  this.canvas.width = 256;
  this.canvas.height = 512;
  this.canvas.style.display = 'none';
  document.body.appendChild(this.canvas);

  // 2. 获取2D上下文（用于绘制三角形实体）
  this.ctx = this.canvas.getContext('2d');

  // 3. 初始化流体动力学
  this.fluidDynamics = new FluidDynamics(this.canvas, {
    width: this.canvas.width,
    height: this.canvas.height,
    curl: 0,                      // 禁用涡流（层流）
    velocityDissipation: 0.999999, // 速度消散率（极高值）
    dyeDissipation: 0.999999,      // 染料消散率（极高值）
    pressureIterations: 20        // 压力迭代次数
  });

  // 4. 创建Three.js纹理
  this.texture = new THREE.CanvasTexture(this.canvas);
  this.texture.minFilter = THREE.LinearFilter;
  this.texture.magFilter = THREE.LinearFilter;
  this.texture.format = THREE.RGBAFormat;
  this.texture.needsUpdate = true;
}
```

**关键参数详解：**

| 参数 | 值 | 说明 | 取值范围 |
|------|-----|------|----------|
| canvas.width | 256 | 画布水平分辨率 | 32-2048 |
| canvas.height | 512 | 画布垂直分辨率 | 32-2048 |
| curl | 0 | 涡度（控制旋涡强度） | 0-50 |
| velocityDissipation | 0.999999 | 速度衰减率 | 0-1 |
| dyeDissipation | 0.999999 | 染料衰减率 | 0-1 |
| pressureIterations | 20 | 压力求解迭代次数 | 1-50 |

**关于 Dissipation（消散率）：**
- 值越高，流体衰减越快
- 0.999999 意味着每一帧保留 0.000001 的流体
- 这使得水流呈现"快速消散"的视觉效果
- 降低此值（如 0.95）会使水流持续更长时间

### 2. 水滴注入 (injectDrops)

```typescript
private injectDrops(): void {
  // ============ 水滴释放参数 ============
  const cx = this.canvas.width / 2;   // 画布中心X = 128
  const topY = 50;                     // 水滴释放Y位置（像素）
  const dropSize = 16;                 // 水滴尺寸（像素）

  // ============ 水滴释放逻辑 ============
  // 定时释放，每隔 dropInterval 秒释放一个
  if (this.time - this.lastDropTime > this.dropInterval) {
    // 释放蓝色水滴
    // setDye(x, y, radius, width, height, color)
    this.fluidDynamics.setDye(
      cx,           // x: 中心X
      topY,         // y: 中心Y
      0,            // radius: 半径
      dropSize,     // width: 宽度
      dropSize * 0.6, // height: 高度（椭圆）
      [0.2, 0.6, 0.9] // color: RGB蓝色
    );

    // 设置初始下落速度
    // setVelocity(x, y, radius, width, height, vx, vy)
    this.fluidDynamics.setVelocity(
      cx, topY, 0,
      dropSize * 0.8,  // 影响范围宽度
      dropSize * 0.4,  // 影响范围高度
      0,               // vx: 水平速度（无）
      500              // vy: 垂直速度（向下500）
    );

    this.lastDropTime = this.time;
  }

  // ============ 重力加速度场 ============
  // 沿水流下落路径施加递增的重力
  for (let y = topY; y < this.canvas.height - 50; y += 15) {
    // 重力随下落距离递增
    const gravity = 80 + (y - topY) * 0.3;
    this.fluidDynamics.setVelocity(cx, y, 0, 15, 8, 0, gravity);
  }
}
```

**setDye 和 setVelocity 参数对比：**

| 参数 | setDye | setVelocity | 说明 |
|------|--------|-------------|------|
| x, y | ✓ | ✓ | 位置 |
| radius | ✓ | ✓ | 半径 |
| width, height | ✓ | ✓ | 影响范围 |
| color | ✓ | - | RGB颜色数组 |
| vx, vy | - | ✓ | 速度分量 |

### 3. 三角形障碍物 (injectTriangleBlocker)

```typescript
private injectTriangleBlocker(): void {
  // ============ 三角形几何参数 ============
  const cx = this.canvas.width / 2;                   // 中心X = 128
  const baseY = this.canvas.height * 0.35;            // 底部Y = 179
  const size = this.triangleSize;                      // 基础尺寸 = 8

  // ============ 尖端位置计算 ============
  // 尖端Y = 底部Y - (尺寸 × 高度系数)
  const tipY = baseY - size * 10;                      // tipY = 179 - 80 = 99
  // 确保尖端不低于最小值
  const actualTipY = Math.max(tipY, 80);               // actualTipY = 99

  // ============ 流体阻塞区域 ============
  // 沿三角形高度方向设置速度为0，阻挡水流通过
  for (let y = actualTipY; y <= baseY; y += 3) {
    // 线性插值计算当前Y处的半宽
    // t=0 时为尖端（最窄），t=1 时为底部（最宽）
    const t = (y - actualTipY) / (baseY - actualTipY);
    const halfWidth = size * 0.6 * (1 + t);

    // 设置速度为0，阻止流体运动
    this.fluidDynamics.setVelocity(cx, y, 0, halfWidth * 2 + 2, 4, 0, 0);
  }

  // ============ 绘制白色实体 ============
  const ctx = this.canvas.getContext('2d');
  if (ctx) {
    ctx.fillStyle = 'rgba(255, 255, 255, 0.9)';
    ctx.beginPath();
    ctx.moveTo(cx, actualTipY);              // 顶点（尖端朝上）
    ctx.lineTo(cx - size * 0.6, baseY);       // 左下
    ctx.lineTo(cx + size * 0.6, baseY);       // 右下
    ctx.closePath();
    ctx.fill();
  }
}
```

**三角形几何示意：**

```
        尖端 (actualTipY)
           /\
          /  \
         /    \
        /      \
       /   ←── halfWidth 递增
      /________\

    底部 (baseY)
```

## 物理模拟原理

### 流体动力学基础

本模块使用 Eulerian 流体模拟方法（基于网格），核心参数：

1. **速度场 (Velocity Field)**
   - 二维向量场，记录每个格点的速度
   - 通过 setVelocity 设置局部速度
   - velocityDissipation 控制速度衰减

2. **染料场 (Dye Field)**
   - 存储颜色信息，用于视觉渲染
   - 通过 setDye 注入染料
   - dyeDissipation 控制染料衰减

3. **压力求解 (Pressure Solve)**
   - 通过压力迭代保持流体不可压缩性
   - pressureIterations 影响模拟精度

### 水流下落物理

```
初始状态：
  位置: (cx=128, topY=50)
  初速度: (0, 500)  // 向下500像素/秒

加速度场：
  for y in [50, 462]:
    gravity(y) = 80 + (y - 50) * 0.3

时间演化：
  t=0:    水滴释放，位于 y=50
  t=0.1:  水滴下落约 50-60 像素
  t=0.2:  水滴接近三角形尖端 (y≈99)
  t=0.3+: 水滴撞击分流，向两侧流动
```

### 撞击分流机制

```
        水流下落
            │
            ↓
    ┌───────────────┐
    │   三角形尖端   │
    │      /\       │
    │     /  \      │
    │    /    \     │
    └───────────────┘
       ↙         ↘
      向左分流   向右分流
```

**分流原因：**
1. 三角形尖端设置速度为0，阻挡直接通过的水流
2. 水流被迫寻找其他路径
3. 由于三角形两侧开放，水流向两侧扩散

## 故障排除

### 问题1：水流没有出现

**可能原因：**
- 画布未正确添加到DOM
- dissipation值过高
- 水滴释放位置不正确

**解决方案：**
```typescript
// 检查画布是否添加到DOM
console.log('Canvas in DOM:', document.body.contains(this.canvas));

// 降低dissipation值
velocityDissipation: 0.95,  // 从0.999999降低
dyeDissipation: 0.95,

// 检查释放位置
console.log('Drop position:', cx, topY);
```

### 问题2：三角形不显示

**可能原因：**
- 三角形颜色与背景相同
- 三角形位置超出画布范围
- Canvas 2D上下文未获取

**解决方案：**
```typescript
// 确保颜色不透明
ctx.fillStyle = 'rgba(255, 255, 255, 1.0)';  // 不使用透明

// 检查三角形位置
console.log('Triangle tip:', actualTipY, 'base:', baseY);
```

### 问题3：性能卡顿

**可能原因：**
- 画布分辨率过高
- 水滴释放过于频繁
- 压力迭代次数过多

**解决方案：**
```typescript
// 降低画布分辨率
this.canvas.width = 128;
this.canvas.height = 256;

// 增大释放间隔
this.dropInterval = 0.1;

// 减少压力迭代
pressureIterations: 10
```
