# 开发进度与函数说明文档

## 1. 项目概述

本项目是一个基于 React + Three.js 的 3D 纸片风格游戏，核心玩法包含**日常行动**和**战斗**两种模式，支持**锁定式**和**自由式**两种射击系统。

### 1.1 核心功能

- **页面管理器**：通过 `gameStore.mode` 控制页面切换，支持菜单、日常、战斗、商店、抽卡、设置 6 种模式
- **双射击系统**：锁定射击（5 条射线检测、1 秒锁定）+ 自由射击（指针锁定、点击发射）
- **子弹池系统**：对象池模式管理子弹，避免频繁创建销毁
- **调试模式**：可视化射线、坐标信息、射击方向等调试信息
- **鼠标锁定控制**：根据模式和射击系统自动控制指针锁定

### 1.2 技术栈

| 技术 | 用途 |
|------|------|
| React 18 | 前端框架 |
| Three.js | 3D 渲染核心 |

#### 动态补偿修正

```typescript
export function getCorrectedNDC(
  canvas: HTMLCanvasElement,
  clientX: number,
  clientY: number,
  camera: THREE.Camera,
  characterPosition: THREE.Vector3,
  baseCompensation: number = 0.3
): CorrectedNDCResult {
  // 1. 计算基础 NDC 坐标
  const ndcX = ((clientX - rect.left) / rect.width) * 2 - 1;
  const ndcY = -((clientY - rect.top) / rect.height) * 2 + 1;

  // 2. 计算相机仰角
  const pitch = getCameraPitch(camera);

  // 3. 距离补偿：远距离（>25米）完全不修正
  const distanceToCamera = camera.position.distanceTo(characterPosition);
  const distanceCompensation = distanceToCamera > 25 ? 0 : Math.min(1.2, 20 / distanceToCamera);

  // 4. 俯角补偿：俯角越大，补偿系数越大
  const pitchCompensation = 1.0 + Math.max(0, pitch) * 0.5;

  // 5. 综合补偿系数
  const totalCompensation = baseCompensation * distanceCompensation * pitchCompensation;
  const correction = -pitch * totalCompensation;

  // 6. 应用修正
  const correctedNDC = new THREE.Vector2(ndcX, ndcY + correction);
  return { raw: rawNDC, corrected: correctedNDC };
}
```

### 4.5 自由式射击 (FreeStyleShootingSystem)

```typescript
private getRaycastHitPoint(): THREE.Vector3 | null {
  const raycaster = new THREE.Raycaster();
  const direction = new THREE.Vector3();
  this.camera.getWorldDirection(direction);
  raycaster.set(this.camera.position, direction);

  if (this.shootableObjects.length > 0) {
    const intersects = raycaster.intersectObjects(this.shootableObjects, true);
    if (intersects.length > 0) {
      return intersects[0].point;
    }
  }

  // 未击中任何物体时，返回射线最远点
  return this.camera.position.clone().add(direction.multiplyScalar(100));
}
```

### 5.3 友方实体 (FriendlyEntity)

**功能**：友方角色（玩家控制或友方AI角色）

**特殊功能**：
- **玩家控制模式**：从 `characterPositionStore` 获取玩家位置
- **AI控制模式**：当 `isPlayerControlled` 为 false 时，跟随玩家
- **角色直面摄像机**：玩家控制的角色始终背对摄像机（直面玩家）

**核心代码**：

```typescript
public update(delta: number): void {
  if (!this.isActive) return;

  if (this.isPlayerControlled) {
    // 从 characterPositionStore 获取玩家位置
    const storePos = characterPositionStore.position;
    this.position.copy(storePos);
    this.mesh.position.copy(this.position);
    this.isMoving = characterPositionStore.isMoving;

    // 让角色直面摄像机
    this.faceToCamera(delta);
  } else {
    super.update(delta);
  }

  if (this.attackCooldown > 0) {
    this.attackCooldown -= delta;
  } else {
    this.attack(delta);
  }
}

private faceToCamera(delta: number): void {
  if (!this.cameraRef || !this.mesh) return;

  const cameraPos = this.cameraRef.position.clone();
  const characterPos = this.mesh.position.clone();

  // 计算从角色指向相机的水平方向
  const toCamera = new THREE.Vector3().subVectors(cameraPos, characterPos);
  toCamera.y = 0;
  toCamera.normalize();

  // 角色背面朝向相机 → 角色的正面方向 = 相机方向的相反数
  const targetDirection = toCamera.clone().negate();

  // 计算目标旋转（只绕Y轴）
  const targetQuat = new THREE.Quaternion().setFromUnitVectors(
    new THREE.Vector3(0, 0, 1),  // 角色默认正面是 +Z
    targetDirection
  );

  // 平滑旋转
  const rotationSpeed = 8.0;
  this.mesh.quaternion.rotateTowards(targetQuat, rotationSpeed * delta);
  this.mesh.quaternion.normalize();
}
```

### 5.4 敌方实体 (EnemyEntity)

**功能**：敌方AI角色，自动追踪玩家

**AI行为**：
- 查找 `isPlayerControlled` 为 true 的角色作为目标
- 计算到目标的水平方向
- 设置移动方向为朝向目标

**核心代码**：

```typescript
protected updateAI(delta: number): void {
  const target = EntityManager.getInstance().getEntitiesByType('character')
    .find(e => (e as CharacterEntity).isPlayerControlled === true) as CharacterEntity;
  if (!target) return;
  const direction = new THREE.Vector3().subVectors(target.position, this.position);
  direction.y = 0;
  if (direction.length() > 0.01) {
    this.moveDirection.copy(direction);
  } else {
    this.moveDirection.set(0, 0, 0);
  }
}
```

### 7.2 核心功能

|---------|---------|---------|
| 子弹 vs 角色 | 子弹与敌人距离 < 半径之和 | 敌人扣血，子弹销毁 |
| 子弹 vs 静态物品 | 子弹与靶子距离 < 半径之和 | 靶子扣血，子弹销毁 |
| 角色 vs 角色 | 玩家与敌人距离 < 半径之和 | 双方扣血 |

```typescript
// src/core/CollisionManager.ts
class GridSpatialPartition {
  private cellSize: number;
  private grid: Map<string, Entity[]>;

  constructor(cellSize: number = 10) {
    this.cellSize = cellSize;
    this.grid = new Map();
  }

  private getCellKey(x: number, y: number, z: number): string {
    const cellX = Math.floor(x / this.cellSize);
    const cellY = Math.floor(y / this.cellSize);
    const cellZ = Math.floor(z / this.cellSize);
    return `${cellX},${cellY},${cellZ}`;
  }

  private getNeighborKeys(cellKey: string): string[] {
    const [cx, cy, cz] = cellKey.split(',').map(Number);
    const neighbors: string[] = [];
    for (let dx = -1; dx <= 1; dx++) {
      for (let dy = -1; dy <= 1; dy++) {
        for (let dz = -1; dz <= 1; dz++) {
          neighbors.push(`${cx + dx},${cy + dy},${cz + dz}`);
        }
      }
    }
    return neighbors;
  }

  public rebuild(entities: Entity[]): void {
    this.clear();
    for (const entity of entities) {
      if (entity.isActive) {
        this.insert(entity);
      }
    }
  }

  public getPotentialCollisions(entity: Entity): Entity[] {
    const key = this.getCellKey(entity.position.x, entity.position.y, entity.position.z);
    const neighborKeys = this.getNeighborKeys(key);
    const candidates: Entity[] = [];

    for (const neighborKey of neighborKeys) {
      const cellEntities = this.grid.get(neighborKey);
      if (cellEntities) {
        candidates.push(...cellEntities);
      }
    }

    return candidates;
  }
}

export class CollisionManager {
  private static instance: CollisionManager;
  private callbacks: Map<string, CollisionCallback> = new Map();
  private spatialPartition: GridSpatialPartition;
  private collisionCooldowns: Map<string, number>;

  private constructor() {
    this.spatialPartition = new GridSpatialPartition(10);
    this.collisionCooldowns = new Map();
  }

  public update(): void {
    const entities = EntityManager.getInstance().getAllEntities();
    this.spatialPartition.rebuild(entities);

    const checkedPairs = new Set<string>();

    for (const entity of entities) {
      if (!entity.isActive) continue;
      const candidates = this.spatialPartition.getPotentialCollisions(entity);

      for (const other of candidates) {
        if (!other.isActive) continue;
        if (entity === other) continue;

        const pairKey = this.getCollisionKey(entity, other);
        if (checkedPairs.has(pairKey)) continue;
        checkedPairs.add(pairKey);

        const dx = entity.position.x - other.position.x;
        const dy = entity.position.y - other.position.y;
        const dz = entity.position.z - other.position.z;
        const distSq = dx * dx + dy * dy + dz * dz;
        const minDist = entity.radius + other.radius;

        if (distSq < minDist * minDist && this.canCollide(entity, other)) {
          this.handleCollision(entity, other);
        }
      }
    }
  }

  private canCollide(a: Entity, b: Entity): boolean {
    const key = this.getCollisionKey(a, b);
    const lastCollision = this.collisionCooldowns.get(key);
    const now = Date.now();

    if (lastCollision && now - lastCollision < 100) {
      return false;
    }

    this.collisionCooldowns.set(key, now);
    return true;
  }
}
```

// src/core/AnimationClip.ts
export class AnimationClip {
  public frames: THREE.Texture[] = [];
  public frameRate: number = 12;
  public loop: boolean = true;
  private currentFrame: number = 0;
  private timeAcc: number = 0;
  
  public update(delta: number): THREE.Texture | null {
    if (this.frames.length === 0) return null;
    this.timeAcc += delta;
    const frameInterval = 1 / this.frameRate;
    if (this.timeAcc >= frameInterval) {
      const steps = Math.floor(this.timeAcc / frameInterval);
      this.currentFrame += steps;
      this.timeAcc -= steps * frameInterval;
      if (this.currentFrame >= this.frames.length) {
        if (this.loop) {
          this.currentFrame %= this.frames.length;
        } else {
          this.currentFrame = this.frames.length - 1;
        }
      }
    }
    return this.frames[this.currentFrame];
  }
}
```

// src/core/PaperAnimator.ts
export class PaperAnimator {
  private frontClip: AnimationClip | null = null;
  private backClip: AnimationClip | null = null;
  private currentClip: AnimationClip | null = null;
  private material: THREE.Material & { map?: THREE.Texture };
  
  constructor(material: THREE.Material & { map?: THREE.Texture }) {
    this.material = material;
  }
  
  public updateDirection(charPos: THREE.Vector3, cameraPos: THREE.Vector3): void {
    const toCamera = new THREE.Vector3().subVectors(cameraPos, charPos);
    toCamera.y = 0;
    toCamera.normalize();
    const forward = new THREE.Vector3(0, 0, 1);
    const isFront = forward.dot(toCamera) > 0;
    const targetClip = isFront ? this.frontClip : this.backClip;
    if (targetClip === this.currentClip) return;
    this.currentClip = targetClip;
    this.currentClip?.reset();
  }
  
  public update(delta: number): void {
    if (!this.currentClip) return;
    const texture = this.currentClip.update(delta);
    if (texture) this.material.map = texture;
  }
}

  public static async loadClip(
    basePath: string,
    frameCount: number,
    startIndex: number = 0,
    frameRate: number = 12,
    loop: boolean = true
  ): Promise<AnimationClip> {
    const loader = new THREE.TextureLoader();
    const promises: Promise<THREE.Texture>[] = [];
    for (let i = 0; i < frameCount; i++) {
      const idx = startIndex + i;
      const path = `${basePath}/${idx}.png`;
      promises.push(loader.loadAsync(path));
    }
    const loaded = await Promise.all(promises);
    return new AnimationClip(loaded, frameRate, loop);
  }
}
```

      setKeysPressed(prev => {
        const newSet = new Set(prev);
        newSet.add(event.key.toLowerCase());
        return newSet;
      });
    };
    
    const handleKeyUp = (event: KeyboardEvent) => {
      setKeysPressed(prev => {
        const newSet = new Set(prev);
        newSet.delete(event.key.toLowerCase());
        return newSet;
      });
    };
    
    window.addEventListener('keydown', handleKeyDown);
    window.addEventListener('keyup', handleKeyUp);
    
    return () => {
      window.removeEventListener('keydown', handleKeyDown);
      window.removeEventListener('keyup', handleKeyUp);
    };
  }, []);
  
  useFrame((_, delta) => {
    // 按键控制动画
    let animationMode: 'front' | 'back' | null = null;
    if (keysPressed.has('w')) {
      animationMode = 'front'; // 按W键播放正面动画
    } else if (keysPressed.has('s')) {
      animationMode = 'back'; // 按S键播放后背动画
    }
    
    // 更新动画
    if (animatorRef.current) {
      if (animationMode) {
        if (animationMode === 'back') {
          const frontCameraPos = characterPos.clone().add(new THREE.Vector3(0, 0, 10));
          animatorRef.current.updateDirection(characterPos, frontCameraPos);
        } else {
          const backCameraPos = characterPos.clone().add(new THREE.Vector3(0, 0, -10));
          animatorRef.current.updateDirection(characterPos, backCameraPos);
        }
      } else {
        animatorRef.current.updateDirection(characterPos, cameraPos);
      }

    }

## 12. 常见问题与解决方案

### 12.1 射击系统问题

**问题**：自由式子弹偏离准心

**解决方案**：
- 使用射线检测获取实际击中点
- 计算从角色头顶到击中点的方向作为子弹方向
- 调整准心偏移量与射线检测匹配

**问题**：锁定式目标丢失

**解决方案**：
- 检查射线是否被遮挡
- 确保目标在相机视野范围内
- 调整锁定取消条件

### 12.2 性能问题

**问题**：对于非常大的地形，计算量可能过大

**解决方案**：
- 使用空间分区技术，只计算角色周围的地形
- 实现 LOD（Level of Detail），根据距离调整地形细分程度
- 减少每帧计算次数，例如每 2 帧计算一次
- 优化地形细分度，减少顶点数量

### 12.3 视觉问题

**问题**：地形明暗过渡不自然

**解决方案**：
- 调整亮度阈值，使过渡更加平滑
- 增加辅助光照强度，提供更柔和的过渡
- 使用后处理效果，如 bloom，增强视觉效果

**问题**：远处地形过暗

**解决方案**：
- 调整远处亮度调整距离
- 增加远处暗部的亮度系数
- 调整环境光强度
