# @bienehito/fluid-dynamics 源码详细学习

## 一、概述

这是一个基于 **WebGL2** 的 2D 流体动力学求解器，核心算法来自 *NVIDIA GPU Gems 38: Fast Fluid Dynamics Simulation on the GPU* （稳定流体算法）。

---

## 二、流体模拟核心算法（Stam 稳定流体）

### 物理背景：Navier-Stokes 方程
```
∂u/∂t + (u·∇)u = -∇p/ρ + ν∇²u + F
∇·u = 0
```

### 算法步骤分解

| 步骤 | 描述 | 对应 Shader |
|------|------|-------------|
| 1. **涡度计算** | 计算旋度（涡量） | `curl` |
| 2. **涡度力** | 添加涡度力（增强旋涡效果） | `vorticity` |
| 3. **散度计算** | 计算速度场散度 | `divergence` |
| 4. **压力求解** | 求解压力泊松方程 | `pressure` |
| 5. **压力梯度** | 用压力修正速度 | `gradientSubtract` |
| 6. **平流** | 速度和染料的传输 | `advection` |

---

## 三、核心数据结构

### 双缓冲纹理（Double FBO）

```javascript
// 所有模拟数据都是双缓冲的
const createDoubleTextureFBO = (width, height, ...) => {
    const read = createTextureFBO();    // 读缓冲
    const write = createTextureFBO();   // 写缓冲
    return {
        read, write,
        swap() { [read, write] = [write, read]; }
    };
};
```

### 纹理类型

| 纹理 | 格式 | 用途 |
|------|------|------|
| `_dye` | RGBA16F | 染料（颜色） |
| `_velocity` | RG16F | 速度场 |
| `_pressure` | R16F | 压力场 |
| `_divergence` | R16F | 散度场 |
| `_curl` | R16F | 旋度（涡量） |

## 四、核心 Shader 详解

### 1. `advection` - 平流（传输）

```glsl
// 核心思想：从当前位置反推，采样过去的颜色
void main() {
    // 根据当前速度，反推 dt 时间前的位置
    vec2 coord = vUv - dt * texture2D(uVelocity, vUv).xy * texelSize;
    
    // 从那里采样并缩放（考虑衰减）
    gl_FragColor = scale * texture2D(uSource, coord);
}
```

### 2. `curl` - 计算旋度

```glsl
void main() {
    // 采样周围 4 个点的速度
    float L = texture2D(uVelocity, vL).y;
    float R = texture2D(uVelocity, vR).y;
    float T = texture2D(uVelocity, vT).x;
    float B = texture2D(uVelocity, vB).x;
    
    // 旋度 = ∇ × u = (R-L) - (T-B)
    float vorticity = R - L - T + B;
    gl_FragColor = vec4(0.5 * vorticity, 0.0, 0.0, 1.0);
}
```

### 3. `divergence` - 计算散度

```glsl
void main() {
    float L = texture2D(uVelocity, vL).x;
    float R = texture2D(uVelocity, vR).x;
    float T = texture2D(uVelocity, vT).y;
    float B = texture2D(uVelocity, vB).y;
    
    // 边界条件：边界上速度反向（像一堵墙）
    vec2 C = texture2D(uVelocity, vUv).xy;
    if (vL.x < 0.0) { L = -C.x; }
    if (vR.x > 1.0) { R = -C.x; }
    
    // 散度 = ∇·u = 0.5 * (R-L + T-B)
    float div = 0.5 * (R - L + T - B);
    gl_FragColor = vec4(div, 0.0, 0.0, 1.0);
}
```

### 4. `pressure` - 压力求解（Jacobi 迭代）

```glsl
void main() {
    float L = texture2D(uPressure, vL).x;
    float R = texture2D(uPressure, vR).x;
    float T = texture2D(uPressure, vT).x;
    float B = texture2D(uPressure, vB).x;
    float C = texture2D(uPressure, vUv).x;
    float divergence = texture2D(uDivergence, vUv).x;
    
    // Poisson 方程迭代：P = (L+R+T+B - div) / 4
    float pressure = (L + R + B + T - divergence) * 0.25;
    gl_FragColor = vec4(pressure, 0.0, 0.0, 1.0);
}
```

### 5. `gradientSubtract` - 压力梯度

```glsl
void main() {
    float L = texture2D(uPressure, vL).x;
    float R = texture2D(uPressure, vR).x;
    float T = texture2D(uPressure, vT).x;
    float B = texture2D(uPressure, vB).x;
    
    // 从速度中减去压力梯度
    vec2 velocity = texture2D(uVelocity, vUv).xy;
    velocity.xy -= vec2(R - L, T - B);
    
    gl_FragColor = vec4(velocity, 0.0, 1.0);
}
```

## 五、完整模拟流程

```javascript
step(dt) {
    // === 步骤 1: 计算旋度 ===
    programs.curl.use({ uVelocity: _velocity.read });
    blit(_curl);

    // === 步骤 2: 涡度力 ===
    programs.vorticity.use({ uVelocity: _velocity.read, uCurl: _curl });
    blit(_velocity.write);
    _velocity.swap();

    // === 步骤 3: 计算散度 ===
    programs.divergence.use({ uVelocity: _velocity.read });
    blit(_divergence);

    // === 步骤 4: 压力求解 (迭代多次) ===
    for (let i = 0; i < pressureIterations; i++) {
        programs.pressure.use({ uDivergence: _divergence, uPressure: _pressure.read });
        blit(_pressure.write);
        _pressure.swap();
    }

    // === 步骤 5: 压力梯度 ===
    programs.gradientSubtract.use({ uPressure: _pressure.read, uVelocity: _velocity.read });
    blit(_velocity.write);
    _velocity.swap();

    // === 步骤 6: 速度平流 ===
    programs.advection.use({ 
        uVelocity: _velocity.read, 
        uSource: _velocity.read, 
        dt, scale: 1-velocityDissipation*dt 
    });
    blit(_velocity.write);
    _velocity.swap();

    // === 步骤 7: 染料平流 ===
    programs.advection.use({ 
        uSource: _dye.read, 
        scale: 1-dyeDissipation*dt 
    });
    blit(_dye.write);
    _dye.swap();
}
```

## 六、API 调用解析

### `setVelocity(x, y, radius, w, h, vx, vy)`

```javascript
setVelocity(x, y, radius, w, h, vx, vy) {
    // 转换到纹理坐标 [0, 1]
    const point = [x / width, y / height];
    const color = [vx * simScale, vy * simScale, 0];
    
    // 使用 splat shader 绘制高斯斑点
    programs.splat.use({
        uSource: _velocity.read,
        point, color,
        transform: computeTransform(radius, w, h)
    });
    
    blit(_velocity.write);
    _velocity.swap();
}
```

### `splat` Shader（高斯斑点）

```glsl
void main() {
    vec2 p = transform * (vUv - point.xy);
    float e = exp(-dot(p, p));  // 高斯函数
    vec3 splat = e * color;
    vec3 base = texture2D(uSource, vUv).xyz;
    
    // 混合：base * (1-e) + splat
    gl_FragColor = vec4(base * (1.0 - e) + splat, 1.0);
}
```

## 七、固体物体交互

### 原理：从压力场中读取受力

```javascript
_updateSolids(dt) {
    for (const solid of solids) {
        // === 步骤 1: 读取固体周围的压力 ===
        gl.readPixels(...); // 读取压力纹理
        
        // === 步骤 2: 积分压力计算受力 ===
        let forceX = 0, forceY = 0;
        for (each direction) {
            const pressure = samplePressure(...);
            forceX += pressure * cos(angle);
            forceY += pressure * sin(angle);
        }
        
        // === 步骤 3: 更新速度和位置 ===
        solid.velocity[0] += forceX / mass;
        solid.velocity[1] += forceY / mass;
        solid.position[0] += solid.velocity[0] * dt;
        solid.position[1] += solid.velocity[1] * dt;
        
        // === 步骤 4: 在速度图上画固体 ===
        this._drawSolid(solid, this._velocity);
    }
}
```

## 八、性能优化要点

### 1. 双缓冲减少写入依赖
```javascript
readFBO -> shader -> writeFBO
swap()
```

### 2. 降低模拟分辨率
```javascript
simScale: 0.5    // 50% 分辨率
dyeScale: 1.0    // 全分辨率渲染
```

### 3. 减少压力迭代
```javascript
pressureIterations: 20  // 够用就行，不需要 50+
```
