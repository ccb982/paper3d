# 流体模拟特效实现文档

## 一、概述

本项目使用 `@bienehito/fluid-dynamics` 库实现了基于物理模拟的流体特效，主要模拟子弹穿过流体时产生的拖尾效果，包括卡门涡街（Kármán vortex street）、马赫环等物理现象。

## 二、核心原理

### 2.1 物理效应模拟

根据流体力学原理，当物体高速穿过流体时，会产生以下特征：

| 物理效应 | 实现方式 | 视觉表现 |
|---------|---------|---------|
| **滞止效应** | 弹头位置速度为零 | 中心白色高光 |
| **流动分离** | 后方施加低压吸引 | 蓝色拖尾核心 |
| **卡门涡街** | 交替方向旋转力 | 两排交错旋涡 |
| **马赫环** | 多层涡旋叠加 | 彩色环状结构 |

## 三、核心实现

### 3.1 纹理生成器 (BulletFluidTexture.ts)

**初始化配置：**

```typescript
this.fluidDynamics = new FluidDynamics(this.canvas, {
  width: 512,           // 模拟宽度
  height: 512,          // 模拟高度
  simScale: 0.8,        // 模拟缩放
  dyeScale: 1.0,        // 染料缩放
  curl: 4,              // 旋转强度
  velocityDissipation: 0.03,  // 速度消散
  dyeDissipation: 0.02        // 染料消散
});
```

**关键参数说明：**

| 参数 | 值 | 作用 |
|-----|---|-----|
| `simScale` | 0.8 | 控制流体模拟的整体缩放 |
| `dyeScale` | 1.0 | 控制染料扩散速度 |
| `curl` | 4 | 控制流体旋转强度，影响涡旋效果 |
| `velocityDissipation` | 0.03 | 速度消散系数，越小流体越持久 |
| `dyeDissipation` | 0.02 | 颜色消散系数，越小颜色越持久 |

### 3.2 帧更新逻辑

每帧执行以下步骤：

#### 步骤1：更新子弹位置

```typescript
// 子弹沿圆形路径运动
this.bulletAngle += this.bulletSpeed;
this.bulletPosition.x = 256 + Math.cos(this.bulletAngle) * 100;
this.bulletPosition.y = 256 + Math.sin(this.bulletAngle) * 100;
```

#### 步骤2：模拟滞止高压区

在弹头位置设置速度为零，模拟高压区：

```typescript
// 中心速度为零，模拟高压区
this.fluidDynamics.setVelocity(x, y, 0, 20, 20, 0, 0);
// 注入淡白色染料
this.fluidDynamics.setDye(x, y, 0, 15, 15, [1, 1, 0.9]);
```

#### 步骤3：模拟低压吸引区

在弹头后方施加向后的速度：

```typescript
const backwardAngle = this.bulletAngle + Math.PI;
const suctionX = x + Math.cos(backwardAngle) * 30;
const suctionY = y + Math.sin(backwardAngle) * 30;
const vx = Math.cos(backwardAngle) * 200;
const vy = Math.sin(backwardAngle) * 200;

this.fluidDynamics.setVelocity(suctionX, suctionY, 0, 60, 60, vx, vy);
this.fluidDynamics.setDye(suctionX, suctionY, 0, 40, 40, [0.1, 0.3, 1.0]);
```

#### 步骤4：生成卡门涡街

在尾部路径上生成交替方向的涡旋：

```typescript
const trailDistance = 50;
const trailX = x + Math.cos(backwardAngle) * trailDistance;
const trailY = y + Math.sin(backwardAngle) * trailDistance;

// 交替方向的涡旋
const alternating = Math.sin(this.vortexPhase) > 0 ? 1 : -1;

// 生成一对旋涡
const vortex1X = trailX + Math.cos(backwardAngle + Math.PI/2) * vortexOffset;
const vortex1Y = trailY + Math.sin(backwardAngle + Math.PI/2) * vortexOffset;
const vortex2X = trailX - Math.cos(backwardAngle + Math.PI/2) * vortexOffset;
const vortex2Y = trailY - Math.sin(backwardAngle + Math.PI/2) * vortexOffset;

// 对两个涡旋施加旋转力
this.fluidDynamics.setVelocity(vortex1X, vortex1Y, 0, 40, 40, 
  -alternating * vortexStrength, alternating * vortexStrength);
this.fluidDynamics.setVelocity(vortex2X, vortex2Y, 0, 40, 40, 
  alternating * vortexStrength, -alternating * vortexStrength);

// 注入彩色染料
this.fluidDynamics.setDye(vortex1X, vortex1Y, 0, 30, 30, [1.0, 0.1, 0.5]); // 粉色
this.fluidDynamics.setDye(vortex2X, vortex2Y, 0, 30, 30, [0.1, 0.9, 0.8]); // 青色
```

### 3.3 特效集成 (BulletFluidEffect.ts)

创建 Three.js 平面并应用流体纹理：

```typescript
export class BulletFluidEffect extends BaseEffect {
  private mesh: THREE.Mesh;
  private material: THREE.MeshBasicMaterial;

  constructor(position: THREE.Vector3, duration: number = 30.0, size: number = 8.0) {
    super(duration);
    
    // 获取纹理
    const texture = TextureManager.getInstance().getTexture('bulletFluid');
    
    // 创建材质
    this.material = new THREE.MeshBasicMaterial({
      map: texture,
      transparent: true,
      blending: THREE.AdditiveBlending,
      opacity: 0.8
    });
    
    // 创建平面
    const geometry = new THREE.PlaneGeometry(size, size);
    this.mesh = new THREE.Mesh(geometry, this.material);
    this.mesh.position.copy(position);
    this.mesh.rotation.x = -Math.PI / 2; // 平放在地面
    
    // 添加到场景
    scene.add(this.mesh);
  }
}
```

## 六、性能优化建议

### 6.1 参数调整

| 优化目标 | 调整方向 | 影响 |
|---------|---------|-----|
| 提高性能 | 降低 `simScale` | 模拟精度降低 |
| 提高性能 | 降低 `pressureIterations` | 流体精度降低 |
| 增强效果 | 提高 `curl` | 涡旋更明显 |
| 延长拖尾 | 降低 `velocityDissipation` | 性能开销增加 |

### 6.2 资源管理

```typescript
// 确保在特效结束时释放资源
public dispose(): void {
  this.texture.dispose();
  if (this.canvas.parentNode) {
    this.canvas.parentNode.removeChild(this.canvas);
  }
}
```

## 九、总结

本实现通过程序化手段模拟了子弹穿过流体时的物理效应，核心技术点包括：

1. **卡门涡街模拟**：通过交替方向的旋转力实现标志性的两排交错旋涡
2. **滞止与低压区**：通过速度场设置模拟流体动力学特征
3. **多层结构**：通过多组涡旋叠加实现马赫环效果
4. **纹理集成**：将流体模拟结果作为动态纹理应用到场景对象

这种实现方式在保证视觉效果的同时，避免了复杂的 CFD（计算流体力学）计算，适合实时游戏场景使用。
